🗄️ Introduction to Amazon S3

🎯 What is S3?

Amazon S3 (Simple Storage Service) is an object storage service offering 99.999999999% (11 nines) of durability. Store any amount of data — from bytes to petabytes — and retrieve it from anywhere.

📋 Key Features

  • Unlimited storage: No capacity to manage
  • Object size: 0 bytes to 5 TB per object
  • Global namespace: Bucket names unique across all AWS accounts
  • Static website hosting: Host HTML/CSS/JS directly from S3
  • Versioning: Keep multiple versions of every object

📌 Use Cases

  • Backup and archive
  • Data lake for analytics
  • Static web content and media hosting
  • Software delivery / artifact storage

🎯 Summary

S3 is the backbone of AWS storage. Think of it as an infinitely scalable hard drive accessible via REST API.
