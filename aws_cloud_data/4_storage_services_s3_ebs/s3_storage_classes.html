📦 S3 Storage Classes

🎯 Why Storage Classes?

Different data has different access patterns. S3 storage classes let you optimize cost vs. access speed vs. durability.

📋 Storage Classes Overview

  • S3 Standard: Frequently accessed data. High durability + availability. Default class.
  • S3 Intelligent-Tiering: Automatically moves data between tiers based on access patterns. Best for unpredictable access.
  • S3 Standard-IA: Infrequent access but need instant retrieval. Lower storage cost, retrieval fee.
  • S3 One Zone-IA: IA data stored in a single AZ. 20% cheaper than Standard-IA, lower resilience.
  • S3 Glacier Instant Retrieval: Archive data, millisecond retrieval.
  • S3 Glacier Flexible Retrieval: Archive data, retrieval in minutes to hours.
  • S3 Glacier Deep Archive: Cheapest storage, retrieval in 12 hours. For long-term archives.

🔄 S3 Lifecycle Policies

Automate transitioning objects between storage classes (e.g., move to Glacier after 90 days).

🎯 Summary

Use Standard for active data, Intelligent-Tiering for unknown patterns, Glacier for archiving. Lifecycle policies automate the transitions.
