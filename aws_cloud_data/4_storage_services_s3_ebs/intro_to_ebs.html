💾 Introduction to EBS

🎯 What is EBS?

Amazon EBS (Elastic Block Store) provides block storage volumes that attach to EC2 instances like a virtual hard drive. Persist independently from the EC2 instance lifecycle.

📋 Key Features

  • Attach to one EC2 instance at a time (same AZ)
  • Snapshots: Point-in-time backups stored in S3, can replicate across regions
  • Encrypt volumes at rest with AWS KMS
  • Resize volumes without downtime

📌 EBS vs Instance Store

  • EBS: Persistent, survives stop/start/reboot of instance
  • Instance Store: Temporary (ephemeral), lost when instance stops/terminates. Very high I/O, good for caches/scratch space

🎯 Summary

EBS = persistent virtual hard drive for EC2. Always take snapshots before modifying or terminating critical instances.
