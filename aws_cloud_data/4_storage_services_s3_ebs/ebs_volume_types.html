📊 EBS Volume Types

🎯 SSD-Backed Volumes

  • gp3 (General Purpose SSD): Baseline 3,000 IOPS, up to 16,000 IOPS. Best overall choice for most workloads. Default boot volume.
  • io2 Block Express (Provisioned IOPS SSD): Ultra-high performance, up to 256,000 IOPS. Best for critical databases (Oracle, SQL Server) requiring consistent IOPS.

🎯 HDD-Backed Volumes

  • st1 (Throughput Optimized HDD): High throughput for sequential workloads. Big data, data warehouses, log processing. Cannot be boot volume.
  • sc1 (Cold HDD): Lowest cost, infrequently accessed data. Cannot be boot volume.
