🗄️ RDS Database Engines

📋 Supported Engines

  • Amazon Aurora: AWS-built MySQL/PostgreSQL-compatible. 5x faster than MySQL, 3x faster than PostgreSQL. Recommended for most new workloads.
  • MySQL: Most popular open-source RDBMS. Widely used for web apps.
  • PostgreSQL: Advanced open-source RDBMS, great for complex queries and JSON support.
  • MariaDB: MySQL fork. Similar to MySQL.
  • Oracle: Enterprise-grade. Bring-your-own-license or license-included.
  • Microsoft SQL Server: For .NET and Windows workloads.

🌟 Aurora Highlights

  • Auto-scaling storage up to 128 TB
  • Aurora Serverless v2: Auto-scales based on load, billed per ACU (Aurora Capacity Unit)
  • Global Database: Low-latency reads across multiple regions

🎯 Summary

Default to Amazon Aurora for new RDS workloads. Use MySQL/PostgreSQL for portability and cost on smaller workloads.
