🚀 Creating and Deploying Lambda Functions

🛠️ Creating a Function (Console)

  1. Go to Lambda → Create Function
  2. Choose runtime (Python, Node.js, Java, etc.)
  3. Set execution role (IAM Role with needed permissions)
  4. Write or upload your code
  5. Set timeout, memory, environment variables
  6. Test with a sample event

📦 Deployment Options

  • Console editor: Good for small functions and learning
  • ZIP upload: For functions with dependencies
  • Container image: Up to 10 GB, great for ML/complex dependencies
# Python Lambda handler
def lambda_handler(event, context):
    name = event.get('name', 'World')
    return {
        'statusCode': 200,
        'body': f'Hello, {name}!'
    }

🎯 Summary

The lambda_handler(event, context) is your entry point. Event carries the trigger payload; context provides runtime info. Return a response dict for API Gateway integration.
