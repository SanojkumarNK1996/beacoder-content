🔧 AWS CodeCommit, CodeBuild, CodeDeploy

📦 AWS CodeCommit

Fully managed Git repository service (like GitHub, but in AWS). Private repos, integrated with IAM for access control. Note: AWS is deprecating new customer usage of CodeCommit in favor of GitHub/GitLab integrations.

🏗️ AWS CodeBuild

Fully managed build service: compiles source code, runs tests, and produces deployment artifacts. No build servers to manage. Integrates with GitHub, GitLab, Bitbucket as source.

# buildspec.yml - CodeBuild instructions
version: 0.2
phases:
  install:
    commands: - npm install
  build:
    commands: - npm test - npm run build
artifacts:
  files: - dist/**

🚀 AWS CodeDeploy

Automates application deployments to EC2, Lambda, ECS, or on-premises servers.

  • In-place deployment: Update existing instances
  • Blue/Green deployment: Shift traffic to new environment, instant rollback

🎯 Summary

Together these form the AWS DevOps suite. CodeBuild for CI (test + build), CodeDeploy for CD (deploy). Most teams use GitHub Actions or Jenkins with CodeDeploy.
