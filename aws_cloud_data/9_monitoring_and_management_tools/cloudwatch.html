📊 AWS CloudWatch

🎯 What is CloudWatch?

Amazon CloudWatch is AWS's native observability service for monitoring resources and applications. Collect metrics, logs, and events — then set alarms and automate responses.

📋 Key Capabilities

  • Metrics: Automated metrics for all AWS services (CPU, network, request count)
  • Custom Metrics: Publish your own app metrics via PutMetricData API
  • Alarms: Trigger actions (SNS, Auto Scaling) when metrics cross thresholds
  • Logs: Collect and search application logs
  • Dashboards: Visual overview of your infrastructure

📌 Retention

  • Metrics stored for 15 months
  • Log retention configurable: 1 day to 10 years

🎯 Summary

CloudWatch is the unified observability hub for AWS. Set up alarms from day one — don't wait for an outage to discover your app is broken.
