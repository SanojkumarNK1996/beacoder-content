🌍 Amazon Route 53

🎯 What is Route 53?

Amazon Route 53 is AWS's highly available and scalable Domain Name System (DNS) web service. DNS translates human-readable domain names (example.com) to IP addresses.

📋 Key Features

  • Domain registration: Buy and manage domain names directly in Route 53
  • DNS routing: Route traffic to EC2, ELB, S3, CloudFront, or external endpoints
  • Health checks: Monitor endpoints and route away from unhealthy ones

🗺️ Routing Policies

  • Simple: Route to single resource
  • Failover: Active-passive HA (route to secondary if primary fails)
  • Geolocation: Route based on user's location
  • Latency-based: Route to the region with lowest latency
  • Weighted: Split traffic by percentage (A/B testing, blue-green deployments)

🎯 Summary

Route 53 is the DNS foundation for AWS-hosted apps. Use failover routing for disaster recovery, latency routing for global performance.
