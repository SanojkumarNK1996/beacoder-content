⚡ Amazon CloudFront CDN

🎯 What is CloudFront?

Amazon CloudFront is a Content Delivery Network (CDN) that delivers content with low latency by caching it at 400+ edge locations globally, close to end users.

📋 How It Works

  1. User requests content (image, video, web page)
  2. Request goes to nearest CloudFront edge location
  3. If cached (cache hit): served immediately from edge
  4. If not cached (cache miss): CloudFront fetches from origin (S3, EC2, ELB), caches it, then serves

📌 Use Cases

  • Static asset delivery (images, JS, CSS) from S3
  • Video streaming at scale
  • DDoS protection (integrated with AWS Shield)
  • API acceleration for REST/GraphQL APIs
  • HTTPS for S3-hosted static sites

🎯 Summary

Put CloudFront in front of S3 or your web app for: faster global delivery, reduced origin load, and built-in DDoS protection.
