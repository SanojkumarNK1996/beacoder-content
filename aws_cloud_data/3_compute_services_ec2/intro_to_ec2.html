🖥️ Introduction to EC2

🎯 What is EC2?

Amazon EC2 (Elastic Compute Cloud) provides resizable virtual servers (instances) in the cloud. It's the most fundamental AWS compute service.

📋 Key Concepts

  • Instance: A virtual machine running on AWS hardware
  • AMI (Amazon Machine Image): Template containing the OS and software for your instance
  • Instance store: Temporary storage tied to the instance lifecycle
  • EBS (Elastic Block Store): Persistent storage attached to EC2

🔧 EC2 Features

  • Launch in seconds, stop/terminate when done
  • Choose OS (Linux, Windows, macOS)
  • Attach/detach storage volumes
  • Assign elastic IP for static addressing

🎯 Summary

EC2 is "rent a computer." You control the OS, software, network config. It's the building block for most AWS architectures.
