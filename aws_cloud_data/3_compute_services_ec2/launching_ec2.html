🚀 Launching and Managing EC2

🛠️ Launch Steps (Console)

  1. Navigate to EC2 → Launch Instance
  2. Choose an AMI (Amazon Linux 2, Ubuntu, Windows)
  3. Choose an instance type (t2.micro for free tier)
  4. Configure instance details (VPC, subnet, IAM role)
  5. Add storage (EBS volume)
  6. Configure Security Group (firewall rules)
  7. Review and launch with a key pair for SSH access

🔧 Managing Instances

  • Stop: Pause the instance (keep EBS, stop billing for compute)
  • Start: Resume from stopped state
  • Terminate: Delete instance and (optionally) its storage
  • Reboot: Restart OS, instance ID stays the same
# SSH into EC2
ssh -i my-key.pem ec2-user@<public-ip>

🎯 Summary

Launching EC2 takes under 2 minutes. Remember: stop instances when not in use to save costs. Terminate when permanently done.
