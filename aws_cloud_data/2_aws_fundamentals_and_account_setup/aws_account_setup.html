🔑 AWS Account Setup

🚀 Creating an AWS Account

  1. Go to aws.amazon.com and click "Create an AWS Account"
  2. Enter email address and account name
  3. Provide payment information (Free Tier available)
  4. Verify phone number
  5. Choose a support plan (Basic is free)

🔒 Root Account Security

The root account has unrestricted access. Best practices:

  • Enable Multi-Factor Authentication (MFA) on root immediately
  • Never use root for day-to-day operations
  • Create IAM users/roles for all regular access

🏷️ Account Organization

AWS Organizations: Manage multiple AWS accounts under one parent. Apply Service Control Policies (SCPs) across accounts.

🎯 Summary

Secure your root account with MFA immediately after creation, then create least-privilege IAM users for all regular access.
