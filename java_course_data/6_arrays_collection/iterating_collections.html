9. Iterating Collections (Loops, For-each)

Goal: Learn how to access and process each element in a collection using loops and the enhanced for-each loop.

Why Iterate Collections?

Iteration allows you to perform operations (like printing, summing, or modifying) on each element of a collection. Java provides multiple ways to loop through collections such as for loops, for-each loops, and Iterator.

Using For Loop

The traditional for loop is used when you need index-based access, like with an ArrayList. 


// Example: Iterating with a for loop
import java.util.ArrayList;

public class ForLoopExample {
    public static void main(String[] args) {
        ArrayList names = new ArrayList<>();
        names.add("Alice");
        names.add("Bob");
        names.add("Charlie");

        for (int i = 0; i < names.size(); i++) {
            System.out.println("Name: " + names.get(i));
        }
    }
}

Using For-each Loop

The enhanced for-each loop simplifies iteration when you don’t need the index. It works with all collections and arrays. 


// Example: Iterating with for-each
import java.util.HashSet;

public class ForEachExample {
    public static void main(String[] args) {
        HashSet fruits = new HashSet<>();
        fruits.add("Apple");
        fruits.add("Banana");
        fruits.add("Mango");

        for (String fruit : fruits) {
            System.out.println("Fruit: " + fruit);
        }
    }
}

Using Iterator

The Iterator interface provides a safe way to traverse collections, especially when you need to remove elements during iteration. 


// Example: Iterating with Iterator
import java.util.ArrayList;
import java.util.Iterator;

public class IteratorExample {
    public static void main(String[] args) {
        ArrayList numbers = new ArrayList<>();
        numbers.add(10);
        numbers.add(20);
        numbers.add(30);

        Iterator it = numbers.iterator();
        while (it.hasNext()) {
            int num = it.next();
            System.out.println("Number: " + num);
        }
    }
}

Key Points

  • For loop: Best when you need index-based access.
  • For-each loop: Best for simple reading of elements.
  • Iterator: Useful for safe element removal during iteration.
  • All these methods can be used with any class from the Java Collections Framework.
