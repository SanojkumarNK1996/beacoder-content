6. Arrays and Collections

Goal: Handling multiple data efficiently.

Arrays

An array is a collection of elements of the same type stored in contiguous memory locations. It allows fast access using an index and is useful when you need to store multiple values of the same kind.

Single-dimensional Arrays

A single-dimensional array is a simple list of elements.


// Example: Single-dimensional array in Java
public class SingleArrayExample {
    public static void main(String[] args) {
        int[] numbers = {10, 20, 30, 40};

        // Access elements
        System.out.println("First element: " + numbers[0]);

        // Modify an element
        numbers[2] = 35;

        // Iterate through the array
        for (int num : numbers) {
            System.out.println(num);
        }
    }
}

Multi-dimensional Arrays

A multi-dimensional array is an array of arrays, often used for tables or grids.


// Example: Multi-dimensional array in Java
public class MultiArrayExample {
    public static void main(String[] args) {
        int[][] matrix = {
            {1, 2, 3},
            {4, 5, 6},
            {7, 8, 9}
        };

        // Access an element
        System.out.println("Element at [1][2]: " + matrix[1][2]);

        // Print all elements
        for (int i = 0; i < matrix.length; i++) {
            for (int j = 0; j < matrix[i].length; j++) {
                System.out.print(matrix[i][j] + " ");
            }
            System.out.println();
        }
    }
}

Key Points

  • Arrays store multiple values of the same type.
  • Single-dimensional arrays represent lists.
  • Multi-dimensional arrays represent tables or grids.
  • Array indices start from 0.
