8. Collections Framework

Goal: Learn how to store and manipulate groups of objects efficiently using Java Collections.

What is the Collections Framework?

The Collections Framework in Java provides classes and interfaces to handle groups of objects such as lists, sets, and maps. It helps manage data dynamically — unlike arrays, collections can grow or shrink as needed.

ArrayList

ArrayList is a dynamic array that can grow and shrink in size. It allows duplicate elements and maintains insertion order. 


// Example: Using ArrayList
import java.util.ArrayList;

public class ArrayListExample {
    public static void main(String[] args) {
        ArrayList names = new ArrayList<>();
        names.add("Alice");
        names.add("Bob");
        names.add("Charlie");

        System.out.println("Names: " + names);
        names.remove("Bob");
        System.out.println("After removal: " + names);
    }
}

LinkedList

LinkedList stores elements in nodes, where each node points to the next. It’s efficient for inserting and removing elements in the middle of the list. 


// Example: Using LinkedList
import java.util.LinkedList;

public class LinkedListExample {
    public static void main(String[] args) {
        LinkedList numbers = new LinkedList<>();
        numbers.add(10);
        numbers.add(20);
        numbers.addFirst(5);
        numbers.addLast(30);

        System.out.println("LinkedList: " + numbers);
    }
}

HashMap

HashMap stores data in key-value pairs. Each key is unique, and it allows fast retrieval based on keys. 


// Example: Using HashMap
import java.util.HashMap;

public class HashMapExample {
    public static void main(String[] args) {
        HashMap ages = new HashMap<>();
        ages.put("Alice", 25);
        ages.put("Bob", 30);
        ages.put("Charlie", 28);

        System.out.println("Age of Bob: " + ages.get("Bob"));
        System.out.println("All entries: " + ages);
    }
}

HashSet

HashSet stores unique elements and does not maintain insertion order. 


// Example: Using HashSet
import java.util.HashSet;

public class HashSetExample {
    public static void main(String[] args) {
        HashSet fruits = new HashSet<>();
        fruits.add("Apple");
        fruits.add("Banana");
        fruits.add("Apple"); // duplicate ignored

        System.out.println("Fruits: " + fruits);
    }
}

Key Points

  • ArrayList – Fast access, slower insertion/removal in the middle.
  • LinkedList – Efficient insertion/removal, slower random access.
  • HashMap – Key-value storage, keys are unique.
  • HashSet – Stores unique elements, no duplicates allowed.
