12. Java 8+ Features (Optional Advanced)

Goal: Learn modern Java features that make code concise, expressive, and functional.

Lambda Expressions

Lambda expressions allow you to write short, functional-style code. They are used to define anonymous methods in a concise way, often with functional interfaces. 


// Example: Lambda expression
import java.util.*;

public class LambdaExample {
    public static void main(String[] args) {
        List names = Arrays.asList("John", "Alice", "Bob");

        // Traditional loop
        for (String name : names) {
            System.out.println(name);
        }

        // Using lambda
        names.forEach(n -> System.out.println("Hello, " + n));
    }
}

Syntax: (parameters) -> expression

Streams API

The Streams API provides a powerful way to process collections using a functional approach. You can filter, map, and reduce data in a clean, declarative style. 


// Example: Using Streams API
import java.util.*;
import java.util.stream.*;

public class StreamsExample {
    public static void main(String[] args) {
        List numbers = Arrays.asList(1, 2, 3, 4, 5, 6);

        List even = numbers.stream()
                                    .filter(n -> n % 2 == 0)
                                    .collect(Collectors.toList());

        System.out.println("Even numbers: " + even);
    }
}

Common Stream operations: filter(), map(), forEach(), collect(), reduce().

Optional Class

The Optional class helps prevent NullPointerException by representing values that might be missing. It provides safe methods to access data that may or may not exist. 


// Example: Optional class
import java.util.Optional;

public class OptionalExample {
    public static void main(String[] args) {
        Optional name = Optional.ofNullable(null);

        // Using isPresent
        if (name.isPresent()) {
            System.out.println("Name: " + name.get());
        } else {
            System.out.println("Name not found.");
        }

        // Using orElse
        System.out.println("Result: " + name.orElse("Guest"));
    }
}

Useful methods: of(), ofNullable(), isPresent(), orElse(), ifPresent().

Functional Interfaces

A Functional Interface is an interface with exactly one abstract method. Common examples include Runnable, Callable, Comparator, and Consumer. They work naturally with lambda expressions. 


// Example: Functional interface and lambda
@FunctionalInterface
interface Greeting {
    void sayHello(String name);
}

public class FunctionalInterfaceExample {
    public static void main(String[] args) {
        Greeting g = n -> System.out.println("Hello, " + n + "!");
        g.sayHello("Alice");
    }
}

Built-in functional interfaces: Consumer, Supplier, Predicate, Function.

Key Points

  • Lambda expressions make code shorter and cleaner.
  • Streams API simplifies collection processing.
  • Optional reduces null-check errors.
  • Functional interfaces enable functional programming in Java.
