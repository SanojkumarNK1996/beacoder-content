Branching Statements in Java

Branching statements are used to change the normal flow of a program. They allow the program to jump to a different part of the code based on certain conditions.

In Java, the main branching statements are break, continue, and return.

1. break Statement

The break statement is used to stop a loop or a switch statement immediately. When break executes, control comes out of the loop or switch block. 


// Example: using break in a loop
for (int i = 1; i <= 10; i++) {
    if (i == 5) {
        break; // stop the loop when i is 5
    }
    System.out.println(i);
}

This loop prints numbers 1 to 4, then stops when i becomes 5.

2. continue Statement

The continue statement skips the current iteration of a loop and jumps to the next one. It does not stop the entire loop — only skips the rest of the code for that particular round. 


// Example: using continue in a loop
for (int i = 1; i <= 5; i++) {
    if (i == 3) {
        continue; // skip when i is 3
    }
    System.out.println(i);
}

This will print 1, 2, 4, and 5 — skipping 3 because of the continue statement.

3. return Statement

The return statement is used to exit from a method. It can also send a value back to the method that called it. 


// Example: using return in a method
public class Example {
    static int add(int a, int b) {
        return a + b; // sends the result back
    }

    public static void main(String[] args) {
        int sum = add(5, 3);
        System.out.println("Sum: " + sum);
    }
}

Here, the add() method uses return to send the sum back to the main method.

Summary

  • break – exits a loop or switch completely.
  • continue – skips to the next iteration of a loop.
  • return – exits a method, optionally sending a value back.

Branching statements give you control over how your program flows, helping you manage loops and methods effectively.
