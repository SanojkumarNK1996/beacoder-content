Conditional Statements in Java

Conditional statements are used to make decisions in a Java program. They help the program choose different actions based on certain conditions. A condition is usually a logical expression that can be true or false.

Think of it like real life — if it’s raining, you take an umbrella. If it’s not, you don’t. In Java, we use conditional statements to express such choices.

1. if Statement

The if statement checks a condition. If the condition is true, the block of code inside if runs. If not, it is skipped. 


// Example
int number = 10;

if (number > 5) {
    System.out.println("Number is greater than 5");
}

Here, since number > 5 is true, the message will be printed.

2. if-else Statement

The if-else statement gives two possible paths. If the condition is true, one block runs; if false, the other block runs. 


// Example
int number = 3;

if (number > 5) {
    System.out.println("Number is greater than 5");
} else {
    System.out.println("Number is less than or equal to 5");
}

Here, the condition is false, so the else part executes.

3. if-else-if Ladder

Sometimes, you have multiple conditions to check. In that case, you can use an if-else-if ladder. 


// Example
int marks = 75;

if (marks >= 90) {
    System.out.println("Grade: A");
} else if (marks >= 75) {
    System.out.println("Grade: B");
} else if (marks >= 50) {
    System.out.println("Grade: C");
} else {
    System.out.println("Grade: Fail");
}

The program checks each condition from top to bottom and runs the first true one.

4. switch-case Statement

The switch statement is used when you have many possible values for a variable. It compares the variable against several case labels and runs the matching one. 


// Example
int day = 3;

switch (day) {
    case 1:
        System.out.println("Monday");
        break;
    case 2:
        System.out.println("Tuesday");
        break;
    case 3:
        System.out.println("Wednesday");
        break;
    default:
        System.out.println("Invalid day");
}

Here, since day is 3, it prints "Wednesday". The break statement stops the switch from checking other cases. If no case matches, the default block runs.

Summary

  • if – runs code if condition is true.
  • if-else – runs one block for true, another for false.
  • if-else-if – checks multiple conditions in sequence.
  • switch-case – picks one block based on variable value.

Conditional statements make programs intelligent by letting them make decisions.
