🔁 Loops in Java

1. 🎯 Why This Exists (Indian Real-Life Example)

  • In a Swiggy delivery route, the rider visits multiple houses one by one.
  • In an ATM cash counting machine, notes are counted repeatedly until bundle ends.
  • Whenever you repeat the same action many times, in Java you use a loop.

2. 📖 What Are Loops?

  • Loops are used to repeat a block of code multiple times.
  • Instead of writing the same statement again and again, you write it once inside a loop.
  • Main loops in Java: for, while, do-while.

3. 🧾 Syntax / Basic Structure ⭐

// for loop
for (initialization; condition; update) {
  // repeated code
}

// while loop
while (condition) {
  // repeated code
}

// do-while loop
do {
  // repeated code
} while (condition);

4. 🔍 How It Works (Step-by-Step Execution) ⭐

for Loop Example

for (int i = 1; i <= 5; i++) {
  System.out.println(i);
}
  1. Initialization: int i = 1 runs once.
  2. Condition check: is i <= 5? If true → run body.
  3. Update: i++ after body.
  4. Repeats until condition becomes false; prints 1 to 5.

while Loop Example

int i = 1;
while (i <= 5) {
  System.out.println(i);
  i++;
}
  • Condition is checked before each iteration.
  • If it is false at the beginning, loop may run 0 times.

do-while Loop Example

int i = 1;
do {
  System.out.println(i);
  i++;
} while (i <= 5);
  • Body executes first, then condition is checked.
  • So it runs at least once, even if condition is false initially.

5. ✅ When to Use Which Loop

  • for – when number of iterations is known (e.g., print 1–100, loop through array).
  • while – when you don’t know how many times it will repeat (e.g., read input until user exits).
  • do-while – when you want code to run at least once (e.g., show menu at least once).

6. ⛔ When NOT to Use / Be Careful

  • Avoid loops without proper update logic (may cause infinite loop).
  • Don’t put too much code inside loops; separate into methods if logic is big.

7. 🧠 Minimum You MUST Remember

  • Every loop needs: start, condition, update.
  • for/while may run 0 or more times; do-while runs at least once.
  • Be careful with the condition; it decides how long loop runs.

8. ⚠️ Common Mistakes

  • Forgetting to change the loop variable → infinite loop.
  • Using i <= length instead of i < length when iterating arrays.
  • Off-by-one errors (starting from 0 or 1 incorrectly).
