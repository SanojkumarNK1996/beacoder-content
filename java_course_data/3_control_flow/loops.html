Loops in Java

Loops are used in Java to repeat a block of code multiple times. Instead of writing the same code again and again, we can use a loop to run it automatically.

For example, if you want to print numbers from 1 to 5, you can use a loop instead of writing 5 separate print statements.

1. for Loop

The for loop is used when you know exactly how many times you want to repeat something. It has three parts inside the parentheses:

  • Initialization – runs once before the loop starts.
  • Condition – checked before each loop run.
  • Update – runs after each iteration.

// Example: print numbers from 1 to 5
for (int i = 1; i <= 5; i++) {
    System.out.println(i);
}

Here, i starts at 1, the loop continues while i <= 5, and after each run, i increases by 1. This prints numbers 1 to 5.

2. while Loop

The while loop is used when you don’t know in advance how many times the code should repeat. It keeps running as long as the condition is true. 


// Example: print numbers from 1 to 5
int i = 1;

while (i <= 5) {
    System.out.println(i);
    i++;
}

The condition is checked first. If it’s true, the code inside runs. If it’s false at the start, the loop won’t run even once.

3. do-while Loop

The do-while loop is similar to the while loop, but with one key difference — it runs the code at least once, even if the condition is false. 


// Example: print numbers from 1 to 5
int i = 1;

do {
    System.out.println(i);
    i++;
} while (i <= 5);

Here, the code inside do runs first, then the condition is checked. That’s why it always executes at least once.

Summary

  • for loop – use when you know how many times to repeat.
  • while loop – use when the number of repeats is unknown.
  • do-while loop – similar to while, but runs once before checking the condition.

Loops make programs shorter and smarter by allowing repetition without manual effort.
