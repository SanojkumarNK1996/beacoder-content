✍️ Your First Java Program – Hello World

1. 🎯 Why This Exists (Indian Real-Life Example)

  • Like writing your first WhatsApp message to understand how chat works, "Hello World" is your first message to the computer.
  • It helps you test that:
    • Java is installed correctly.
    • Your IDE/terminal is working.
    • You understand basic Java structure.

2. 📖 What Is This Program?

  • A very small Java program that prints a line of text on the screen.
  • It shows you:
    • How to create a class.
    • Where the program starts (main method).
    • How to display output.

3. 🧾 Syntax / Basic Structure ⭐

Type this code in a file named HelloWorld.java:

public class HelloWorld {
  public static void main(String[] args) {
    System.out.println("Hello, World!");
  }
}
  • public class HelloWorld { ... } – defines a class named HelloWorld.
  • public static void main(String[] args) – starting point of the program.
  • System.out.println(...) – prints text to the console.

4. 🔍 How It Works (Step-by-Step Execution) ⭐

  1. Step 1 – Create the file
    • Open your IDE or a text editor.
    • Create a file named HelloWorld.java.
    • Paste the code above and save.
  2. Step 2 – Compile the code
    • Open terminal/command prompt in the folder where the file is saved.
    • Run: javac HelloWorld.java.
    • This creates HelloWorld.class – the bytecode file.
  3. Step 3 – Run the program
    • Run: java HelloWorld (note: no .class).
    • The JVM loads HelloWorld.class and executes main.
  4. Step 4 – See the output
    • You should see: Hello, World! printed on the screen.

5. ✅ When to Use This Pattern

  • To test if Java + IDE are installed properly.
  • Whenever you create a new project and want to verify setup.
  • As a base template – you can later modify the message, add variables, etc.

6. ⛔ When NOT to Use Only This

  • Once comfortable, don’t stop at "Hello World" – move on to variables, conditions, and loops.
  • Don’t use this structure directly in real projects; it’s only a starting example.

7. 🧠 Minimum You MUST Remember

  • File name and public class name must match (HelloWorld).
  • Your program’s entry point is always: public static void main(String[] args).
  • Use System.out.println() to print output.

8. ⚠️ Common Mistakes

  • Mismatch between file name and class name (e.g., file is hello.java but class is HelloWorld).
  • Missing semicolons ; at the end of statements.
  • Using wrong case: Public instead of public, Main instead of main.
  • Typing java HelloWorld.java instead of java HelloWorld when running.

9. 🎤 Interviewer’s Perspective

Possible questions:

  • "Write a Java program to print 'Hello, World!'"
  • "Explain each part of the Hello World program."
  • "Why must the file name and public class name match?"
  • "What is the role of the main method?"
