Writing Your First Java Program (Hello World)

Now that your Java environment and IDE are ready, it’s time to write your first Java program. The traditional first step in any programming language is creating a simple Hello World program. This helps you understand how Java code works and how to run it.

1. What the Program Does

The goal is simple: display the text Hello, World! on the screen. This small program teaches you the structure of a Java application.

2. Step-by-Step Guide

Step 1: Create a New Java File

In your IDE (Eclipse, IntelliJ, or VS Code), create a new Java project and then create a file named HelloWorld.java.

Step 2: Write the Code

Type the following code inside the file:

public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}

Step 3: Save the File

Make sure to save the file with the same name as the class — HelloWorld.java. In Java, the file name and public class name must match.

Step 4: Compile the Code

If you’re using the terminal or command prompt (instead of an IDE), go to the folder where your file is saved and run:

javac HelloWorld.java

This command compiles the Java code and creates a new file called HelloWorld.class (the bytecode).

Step 5: Run the Program

After compilation, type the following command to run your program:

java HelloWorld

You should see the output:

Hello, World!

3. Understanding the Code

  • public class HelloWorld: Defines a class named HelloWorld. Every Java program must have at least one class.
  • public static void main(String[] args): This is the entry point of every Java program. The program starts running from this method.
  • System.out.println(): Prints text to the screen. The text inside the quotation marks appears as output.

4. Common Mistakes to Avoid

  • Make sure the file name and class name are the same (case-sensitive).
  • Don’t forget the semicolon (;) at the end of statements.
  • Always use double quotes (" ") for text inside System.out.println().

5. Summary

Congratulations! 🎉 You’ve written your first Java program. This simple example teaches you how to:

  • Create a Java file.
  • Compile and run Java code.
  • Understand the basic structure of a Java application.

From here, you can explore variables, data types, and other features to build more powerful Java programs.
