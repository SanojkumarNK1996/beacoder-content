Installing JDK and Setting up IDE

Now that you understand what JDK, JRE, and JVM are, let’s move to the practical part — installing the JDK and setting up an IDE (Integrated Development Environment). These are the tools you’ll use to write and run Java programs easily.

1. Installing JDK (Java Development Kit)

The JDK contains everything needed to write and run Java programs. Follow the steps below to install it:

Step-by-Step Installation:

  1. Download the JDK:

    Go to the Oracle JDK Downloads page or use Eclipse Adoptium (OpenJDK) for a free version.

  2. Install the JDK:

    Run the installer and follow the on-screen instructions. Keep the default options unless you know what you’re changing.

  3. Set Environment Variables:
    • Find your JDK installation path (for example, C:\Program Files\Java\jdk-21).
    • Add this path to your PATH variable and create a JAVA_HOME variable pointing to the same directory.
  4. Verify Installation:

    Open a terminal or command prompt and type:

    java -version
javac -version

    If both commands show a version number, your JDK is installed correctly!

2. Setting up an IDE

An IDE (Integrated Development Environment) helps you write, debug, and run Java programs easily. It provides a nice interface, code suggestions, and tools for faster development. Below are three popular IDEs for Java:

(a) Eclipse

  1. Download Eclipse from the official Eclipse website.
  2. Choose “Eclipse IDE for Java Developers.”
  3. Install and launch Eclipse.
  4. When prompted, select a workspace folder — this is where your Java projects will be stored.

Now you can create a new Java project using File → New → Java Project.

(b) IntelliJ IDEA

  1. Download IntelliJ IDEA from the JetBrains website.
  2. Choose the Community Edition (free version) for learning and basic projects.
  3. Install and open IntelliJ IDEA.
  4. Go to File → New Project → Java and select your JDK.

IntelliJ automatically detects the JDK and helps you manage dependencies easily.

(c) Visual Studio Code (VS Code)

  1. Download and install VS Code.
  2. Open VS Code and go to the Extensions tab (left sidebar).
  3. Search for and install “Extension Pack for Java” by Microsoft.
  4. This will install all required tools — Java support, debugger, and project manager.

After setup, you can create a new Java file, write code, and run it directly inside VS Code.

3. Summary

  • JDK is required to compile and run Java programs.
  • IDEs make coding faster and easier with features like auto-completion and debugging.
  • Eclipse is simple and lightweight, IntelliJ is powerful and smart, and VS Code is flexible and easy to customize.

Once your JDK and IDE are ready, you’re all set to write your first Java program!
