Data Types in Java

In Java, data types define the kind of values a variable can hold. For example, a number, a single character, or a piece of text. Java is a strongly typed language, which means you must declare the type of every variable before using it.

1. What Are Data Types?

Every piece of data in Java has a specific type. This helps the compiler understand how much memory to allocate and what operations can be performed on the data.

Java data types are divided into two main categories:

  • Primitive Types — Basic, built-in data types.
  • Non-Primitive Types — More complex types created using classes.

Primitive Data Types

Primitive types are the simplest and most commonly used data types in Java. They store simple values like numbers, characters, and true/false values. There are 8 primitive data types in Java:

Type Size Example Description
byte 1 byte byte b = 10; Stores small integers from -128 to 127.
short 2 bytes short s = 3000; Stores larger integers from -32,768 to 32,767.
int 4 bytes int num = 100000; Stores whole numbers (most common integer type).
long 8 bytes long bigNum = 10000000000L; Stores very large whole numbers. Use L at the end.
float 4 bytes float price = 10.5f; Stores decimal numbers (single precision). Use f at the end.
double 8 bytes double pi = 3.14159; Stores decimal numbers (double precision).
char 2 bytes char letter = 'A'; Stores a single character, like a letter or digit.
boolean 1 bit boolean isJavaFun = true; Stores true or false values.

Example of Using Primitive Types:

public class PrimitiveExample {
    public static void main(String[] args) {
        int age = 25;
        double salary = 55000.75;
        char grade = 'A';
        boolean isActive = true;
        
        System.out.println("Age: " + age);
        System.out.println("Salary: " + salary);
        System.out.println("Grade: " + grade);
        System.out.println("Active: " + isActive);
    }
}
