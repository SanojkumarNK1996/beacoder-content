Conditionals

Why this exists

  • Conditionals let your program make decisions (e.g., show discount only if amount > 500).

What is this

  • if, else if, else, and switch statements for branching logic.

Basic examples

let score = 75;

if (score >= 90) {
  console.log('Grade A');
} else if (score >= 75) {
  console.log('Grade B');
} else {
  console.log('Grade C');
}

Minimum you must remember

  • Conditions are boolean expressions (true/false).
  • Use === for comparisons to avoid unexpected coercion.

Practice questions

  1. Write a program that checks if a number is positive, negative, or zero.
  2. Use switch to handle different days of the week.
