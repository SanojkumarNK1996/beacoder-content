Functions

Why this exists

  • Functions allow you to reuse logic, like calculating GST or validating input.

What is this

  • Blocks of code that can be called with arguments and may return a value.

Basic examples

function add(a, b) {
  return a + b;
}

const multiply = (a, b) => a * b;

console.log(add(2, 3));      // 5
console.log(multiply(2, 3)); // 6

Minimum you must remember

  • Functions can have parameters and return values.
  • Arrow functions are shorter syntax, but handle this differently.
