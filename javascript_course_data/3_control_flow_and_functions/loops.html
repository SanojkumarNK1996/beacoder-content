Loops

Why this exists

  • Loops help repeat tasks, like iterating over a list of products or students.

What is this

  • for, while, do-while, and for...of/for...in loops.

Basic examples

for (let i = 0; i < 5; i++) {
  console.log(i);
}

let i = 0;
while (i < 5) {
  console.log(i);
  i++;
}

Minimum you must remember

  • Avoid infinite loops by ensuring loop conditions eventually fail.
  • Use for...of for arrays and for...in for object keys (with care).

Practice questions

  1. Print all numbers from 1 to 100 using a loop.
  2. Loop over an array of names and print each one.
