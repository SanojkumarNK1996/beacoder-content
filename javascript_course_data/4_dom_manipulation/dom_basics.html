DOM Basics

Why this exists

  • To let JavaScript read and modify HTML elements dynamically (for example, updating cart count or error messages).

What is this

  • DOM (Document Object Model) is a tree representation of the HTML page.
  • JavaScript can query and update this tree.

Basic examples

// HTML: <p id="msg">Hello</p>

const para = document.getElementById('msg');
para.textContent = 'Hello, JavaScript!';
para.style.color = 'blue';

Minimum you must remember

  • Use document.getElementById, querySelector, or querySelectorAll to find elements.
  • You can change text, attributes, styles, and even create/remove elements.

Practice questions

  1. Select a button by ID and change its text using JavaScript.
  2. Create a new list item and append it to an existing <ul>.
