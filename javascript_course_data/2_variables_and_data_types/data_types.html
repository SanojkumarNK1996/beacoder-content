Data Types

Why this exists

  • To correctly store and process different kinds of values (numbers, text, lists, etc.).

What is this

  • JavaScript has primitive types: string, number, boolean, null, undefined, symbol, bigint.
  • And reference types: objects, arrays, functions.

Examples

let name = 'Asha';       // string
let age = 30;            // number
let isStudent = true;    // boolean
let marks = [90, 85, 88]; // array (object)
let person = { name: 'Asha', city: 'Mumbai' }; // object

Minimum you must remember

  • typeof operator helps check the type.
  • Arrays and functions are objects in JavaScript.

Interviewer perspective

  • What are primitive vs reference types?
  • Result of typeof null?

Practice questions

  1. Declare one variable of each primitive type.
  2. Use typeof on different values and note the results.
