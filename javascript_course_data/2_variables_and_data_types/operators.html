Operators

Why this exists

  • Operators perform operations like addition, comparison, and logical checks.

What is this

  • Arithmetic (+, -, *, /), comparison (==, ===, >, <), logical (&&, ||, !), assignment (=, +=, -=), etc.

Examples

let a = 10;
let b = 3;

console.log(a + b);   // 13
console.log(a % b);   // 1
console.log(a > b);   // true
console.log(a === 10); // true
console.log(a == '10'); // true (type coercion)
console.log(a === '10'); // false (strict compare)

Minimum you must remember

  • == does type coercion, === does strict comparison.
  • Logical operators are used a lot in conditionals.

Interviewer perspective

  • Difference between == and ===?
  • How does the && operator behave?

Practice questions

  1. Write expressions using each arithmetic operator.
  2. Compare '5' and 5 with both == and === and observe results.
