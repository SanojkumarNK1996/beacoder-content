Variables

Why this exists

  • Variables let you store values (like username, cart total, or OTP) and reuse them in your code.

What is this

  • Named containers for data in JavaScript.
  • Declared using var, let, or const.

Basic syntax

let name = 'Sanjay';
const pi = 3.14;
var age = 25;

How it works

  1. When you declare a variable, memory is reserved for it.
  2. You can read or update the value (except const, which cannot be reassigned).

Minimum you must remember

  • Use let and const in modern JavaScript; avoid var in new code.
  • Use const when you don’t plan to reassign the variable.

Common mistakes

  • Using var and facing hoisting and scope issues.
  • Reassigning const variables.

Interviewer perspective

  • Difference between var, let, and const?
  • What is block scope?

Practice questions

  1. Declare variables for your name, age, and city using let/const.
  2. Try to reassign a const and observe the error.
