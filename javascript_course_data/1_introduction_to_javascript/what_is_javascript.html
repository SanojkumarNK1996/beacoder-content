What is JavaScript?

Why this exists (Indian real-life example)

  • Features like live train status, dynamic offers on Flipkart, and interactive forms on banking sites are all powered by JavaScript.

What is this

  • JavaScript is a programming language that runs mainly in the browser and makes web pages interactive.
  • It can also run on servers (Node.js) and other environments.

Basic syntax example

// This shows an alert box
alert('Hello from JavaScript');

// This logs a message in the console
console.log('Welcome to JS');

How it works (high level)

  1. Browser loads HTML and CSS first.
  2. Then it loads and executes JavaScript.
  3. JavaScript can read and modify the page, handle events, and talk to servers.

When to use

  • Validating forms in the browser.
  • Showing/hiding sections, modals, dropdowns.
  • Calling APIs and updating data without reloading the page.

When not to use

  • For content that must always be available (like important text) — keep it in HTML.
  • For security-sensitive checks only on the client (always validate on server too).

Minimum you must remember

  • JavaScript adds behavior to HTML + CSS.
  • It runs in the browser’s JavaScript engine (like V8 in Chrome).

Interviewer perspective

  • What problem does JavaScript solve compared to just HTML/CSS?
  • Where does JavaScript run?
