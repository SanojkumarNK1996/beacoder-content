JavaScript in HTML

Why this exists

  • To connect JavaScript code with web pages, we need a standard way to include scripts in HTML.

What is this

  • Ways to add JavaScript using the <script> tag: inline, internal, and external.

Basic patterns

<!-- Internal script -->
<script>
  console.log('Hello from internal script');
</script>

<!-- External script -->
<script src="app.js"></script>

How it works

  1. The browser parses HTML from top to bottom.
  2. When it sees a <script> tag, it loads and executes that JavaScript.
  3. External scripts are fetched from the given src URL.

Minimum you must remember

  • Prefer external scripts for maintainability.
  • Place scripts at the end of <body> or use defer for better performance.

Common mistakes

  • Running scripts before the DOM is ready, causing null errors.
  • Mixing too much inline JavaScript inside HTML attributes.
