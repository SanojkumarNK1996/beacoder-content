Event Handling

Why this exists

  • All interactivity (button clicks, typing, scrolling) is handled via events in JavaScript.

What is this

  • Listening for events like click, input, submit, keydown and running code in response.

Basic example

// HTML: <button id="btn">Click me</button>

const btn = document.getElementById('btn');

btn.addEventListener('click', function () {
  alert('Button clicked!');
});

Minimum you must remember

  • Use addEventListener instead of inline onclick attributes.
  • Event handlers receive an event object with details.

Practice questions

  1. Show an alert when a user hovers over an element.
  2. Log every key press in an input field to the console.
