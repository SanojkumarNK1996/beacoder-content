Form Interactivity

Why this exists

  • To give instant feedback on forms (showing errors, enabling/disabling buttons, etc.) without reloading the page.

What is this

  • Using JavaScript to validate inputs, show messages, and control form submission.

Basic example

// HTML:
// <form id="loginForm">
//   <input id="email" type="email" />
//   <button type="submit">Login</button>
// </form>

const form = document.getElementById('loginForm');
const emailInput = document.getElementById('email');

form.addEventListener('submit', function (event) {
  if (!emailInput.value) {
    event.preventDefault();
    alert('Email is required');
  }
});

Minimum you must remember

  • Use event.preventDefault() to stop default submission when validation fails.
  • Always also validate on the server.

Practice questions

  1. Build a form that requires both email and password before allowing submit.
  2. Show a success message on valid submission without reloading the page.
