🚀 First Spring Boot Application

🎯 What We'll Build

We'll create a complete Spring Boot web application from scratch - a simple Book Management API that demonstrates all core Spring Boot concepts in action.

🏗️ Building a House Analogy

Creating a Spring Boot application is like building a house:

  • 🏠 Foundation: Spring Boot Starter Dependencies
  • 🏗️ Framework: Main Application Class
  • 📦 Rooms: Controllers, Services, Models
  • 🚪 Entrances: REST Endpoints
  • ⚙️ Utilities: Configuration & Properties

Spring Boot provides the blueprint and tools!

package com.example.bookmanagement.model;

import java.time.LocalDate;

/**
 * Book entity representing a book in our system
 * This is a POJO (Plain Old Java Object) that represents our data model
 */
public class Book {
    
    // Primary fields
    private Long id;
    private String title;
    private String author;
    private String isbn;
    private LocalDate publishDate;
    private Double price;
    private Integer pages;
    private String genre;
    private String description;
    private Boolean available;
    
    // Default constructor (required for JSON serialization)
    public Book() {}
    
    // Parameterized constructor for easy object creation
    public Book(Long id, String title, String author, String isbn, 
                LocalDate publishDate, Double price, Integer pages, 
                String genre, String description, Boolean available) {
        this.id = id;
        this.title = title;
        this.author = author;
        this.isbn = isbn;
        this.publishDate = publishDate;
        this.price = price;
        this.pages = pages;
        this.genre = genre;
        this.description = description;
        this.available = available;
    }
    
    // Getters and Setters (required for JSON serialization/deserialization)
    public Long getId() { return id; }
    public void setId(Long id) { this.id = id; }
    
    public String getTitle() { return title; }
    public void setTitle(String title) { this.title = title; }
    
    public String getAuthor() { return author; }
    public void setAuthor(String author) { this.author = author; }
    
    public String getIsbn() { return isbn; }
    public void setIsbn(String isbn) { this.isbn = isbn; }
    
    public LocalDate getPublishDate() { return publishDate; }
    public void setPublishDate(LocalDate publishDate) { this.publishDate = publishDate; }
    
    public Double getPrice() { return price; }
    public void setPrice(Double price) { this.price = price; }
    
    public Integer getPages() { return pages; }
    public void setPages(Integer pages) { this.pages = pages; }
    
    public String getGenre() { return genre; }
    public void setGenre(String genre) { this.genre = genre; }
    
    public String getDescription() { return description; }
    public void setDescription(String description) { this.description = description; }
    
    public Boolean getAvailable() { return available; }
    public void setAvailable(Boolean available) { this.available = available; }
    
    // toString method for debugging
    @Override
    public String toString() {
        return "Book{" +
                "id=" + id +
                ", title='" + title + '\'' +
                ", author='" + author + '\'' +
                ", isbn='" + isbn + '\'' +
                ", price=" + price +
                ", available=" + available +
                '}';
    }
}

🏪 Step 4: Create the Service Layer

📋 BookService Class (Business Logic)

File: src/main/java/com/example/bookmanagement/service/BookService.java

package com.example.bookmanagement.service;

import com.example.bookmanagement.model.Book;
import org.springframework.stereotype.Service;

import java.time.LocalDate;
import java.util.ArrayList;
import java.util.List;
import java.util.Optional;

/**
 * Service layer for Book operations
 * Contains business logic and data manipulation
 * 
 * @Service annotation marks this as a Spring service component
 * Spring will automatically detect and register this as a bean
 */
@Service
public class BookService {
    
    // In-memory storage for demonstration (in real apps, this would be a database)
    private List<Book> books = new ArrayList<>();
    private Long nextId = 1L;
    
    // Constructor to initialize some sample data
    public BookService() {
        initializeSampleData();
    }
    
    /**
     * Get all books
     * @return List of all books
     */
    public List<Book> getAllBooks() {
        return new ArrayList<>(books); // Return a copy to prevent external modification
    }
    
    /**
     * Get a book by ID
     * @param id The book ID
     * @return Optional containing the book if found, empty otherwise
     */
    public Optional<Book> getBookById(Long id) {
        return books.stream()
                   .filter(book -> book.getId().equals(id))
                   .findFirst();
    }
    
    /**
     * Create a new book
     * @param book The book to create
     * @return The created book with assigned ID
     */
    public Book createBook(Book book) {
        book.setId(nextId++);
        books.add(book);
        System.out.println("📚 Created new book: " + book.getTitle());
        return book;
    }
    
    /**
     * Update an existing book
     * @param id The book ID
     * @param updatedBook The updated book data
     * @return Optional containing the updated book if successful
     */
    public Optional<Book> updateBook(Long id, Book updatedBook) {
        for (int i = 0; i < books.size(); i++) {
            if (books.get(i).getId().equals(id)) {
                updatedBook.setId(id);
                books.set(i, updatedBook);
                System.out.println("📝 Updated book: " + updatedBook.getTitle());
                return Optional.of(updatedBook);
            }
        }
        return Optional.empty();
    }
    
    /**
     * Delete a book by ID
     * @param id The book ID
     * @return true if deleted successfully, false otherwise
     */
    public boolean deleteBook(Long id) {
        boolean removed = books.removeIf(book -> book.getId().equals(id));
        if (removed) {
            System.out.println("🗑️ Deleted book with ID: " + id);
        }
        return removed;
    }
    
    /**
     * Search books by title or author
     * @param query Search query
     * @return List of matching books
     */
    public List<Book> searchBooks(String query) {
        String lowerQuery = query.toLowerCase();
        return books.stream()
                   .filter(book -> 
                       book.getTitle().toLowerCase().contains(lowerQuery) ||
                       book.getAuthor().toLowerCase().contains(lowerQuery))
                   .toList();
    }
    
    /**
     * Get books by availability
     * @param available Whether to get available or unavailable books
     * @return List of books matching availability status
     */
    public List<Book> getBooksByAvailability(boolean available) {
        return books.stream()
                   .filter(book -> book.getAvailable() == available)
                   .toList();
    }
    
    /**
     * Initialize some sample data for demonstration
     */
    private void initializeSampleData() {
        books.add(new Book(nextId++, "The Great Gatsby", "F. Scott Fitzgerald", 
                  "978-0-7432-7356-5", LocalDate.of(1925, 4, 10), 
                  12.99, 180, "Classic Fiction", 
                  "A classic American novel about the Jazz Age", true));
        
        books.add(new Book(nextId++, "To Kill a Mockingbird", "Harper Lee", 
                  "978-0-06-112008-4", LocalDate.of(1960, 7, 11), 
                  14.99, 281, "Literary Fiction", 
                  "A gripping tale of racial injustice and childhood innocence", true));
        
        books.add(new Book(nextId++, "1984", "George Orwell", 
                  "978-0-452-28423-4", LocalDate.of(1949, 6, 8), 
                  13.99, 328, "Dystopian Fiction", 
                  "A dystopian social science fiction novel", false));
        
        System.out.println("📚 Initialized " + books.size() + " sample books");
    }
}

🌐 Step 5: Create the REST Controller

📡 BookController Class (REST API)

File: src/main/java/com/example/bookmanagement/controller/BookController.java

package com.example.bookmanagement.controller;

import com.example.bookmanagement.model.Book;
import com.example.bookmanagement.service.BookService;
import org.springframework.beans.factory.annotation.Autowired;
import org.springframework.http.HttpStatus;
import org.springframework.http.ResponseEntity;
import org.springframework.web.bind.annotation.*;

import java.util.List;
import java.util.Optional;

/**
 * REST Controller for Book Management API
 * Handles HTTP requests and responses
 * 
 * @RestController combines @Controller and @ResponseBody
 * @RequestMapping provides the base URL path for all endpoints
 */
@RestController
@RequestMapping("/api/books")
@CrossOrigin(origins = "*") // Allow cross-origin requests (for frontend development)
public class BookController {
    
    // Inject the BookService dependency
    private final BookService bookService;
    
    // Constructor injection (recommended over @Autowired on field)
    @Autowired
    public BookController(BookService bookService) {
        this.bookService = bookService;
    }
    
    /**
     * GET /api/books - Get all books
     * @return List of all books
     */
    @GetMapping
    public ResponseEntity<List<Book>> getAllBooks() {
        List<Book> books = bookService.getAllBooks();
        return ResponseEntity.ok(books);
    }
    
    /**
     * GET /api/books/{id} - Get a specific book by ID
     * @param id Book ID from URL path
     * @return ResponseEntity with book data or 404 if not found
     */
    @GetMapping("/{id}")
    public ResponseEntity<Book> getBookById(@PathVariable Long id) {
        Optional<Book> book = bookService.getBookById(id);
        
        if (book.isPresent()) {
            return ResponseEntity.ok(book.get());
        } else {
            return ResponseEntity.notFound().build();
        }
    }
    
    /**
     * POST /api/books - Create a new book
     * @param book Book data from request body (JSON)
     * @return ResponseEntity with created book data
     */
    @PostMapping
    public ResponseEntity<Book> createBook(@RequestBody Book book) {
        try {
            Book createdBook = bookService.createBook(book);
            return ResponseEntity.status(HttpStatus.CREATED).body(createdBook);
        } catch (Exception e) {
            return ResponseEntity.badRequest().build();
        }
    }
    
    /**
     * PUT /api/books/{id} - Update an existing book
     * @param id Book ID from URL path
     * @param book Updated book data from request body
     * @return ResponseEntity with updated book data or 404 if not found
     */
    @PutMapping("/{id}")
    public ResponseEntity<Book> updateBook(@PathVariable Long id, @RequestBody Book book) {
        Optional<Book> updatedBook = bookService.updateBook(id, book);
        
        if (updatedBook.isPresent()) {
            return ResponseEntity.ok(updatedBook.get());
        } else {
            return ResponseEntity.notFound().build();
        }
    }
    
    /**
     * DELETE /api/books/{id} - Delete a book
     * @param id Book ID from URL path
     * @return ResponseEntity with appropriate status
     */
    @DeleteMapping("/{id}")
    public ResponseEntity<Void> deleteBook(@PathVariable Long id) {
        boolean deleted = bookService.deleteBook(id);
        
        if (deleted) {
            return ResponseEntity.noContent().build(); // 204 No Content
        } else {
            return ResponseEntity.notFound().build(); // 404 Not Found
        }
    }
    
    /**
     * GET /api/books/search?q={query} - Search books by title or author
     * @param query Search query parameter
     * @return List of matching books
     */
    @GetMapping("/search")
    public ResponseEntity<List<Book>> searchBooks(@RequestParam("q") String query) {
        List<Book> books = bookService.searchBooks(query);
        return ResponseEntity.ok(books);
    }
    
    /**
     * GET /api/books/available - Get available books only
     * @return List of available books
     */
    @GetMapping("/available")
    public ResponseEntity<List<Book>> getAvailableBooks() {
        List<Book> books = bookService.getBooksByAvailability(true);
        return ResponseEntity.ok(books);
    }
    
    /**
     * GET /api/books/unavailable - Get unavailable books only
     * @return List of unavailable books
     */
    @GetMapping("/unavailable")
    public ResponseEntity<List<Book>> getUnavailableBooks() {
        List<Book> books = bookService.getBooksByAvailability(false);
        return ResponseEntity.ok(books);
    }
    
    /**
     * GET /api/books/health - Health check endpoint
     * @return Simple status message
     */
    @GetMapping("/health")
    public ResponseEntity<String> healthCheck() {
        return ResponseEntity.ok("📚 Book Management API is running smoothly!");
    }
}

📊 Step 9: Understanding HTTP Status Codes

✅ Success Codes

  • 200 OK: GET requests successful
  • 201 Created: POST request created new resource
  • 204 No Content: DELETE request successful

❌ Error Codes

  • 400 Bad Request: Invalid JSON or request data
  • 404 Not Found: Resource doesn't exist
  • 500 Internal Server Error: Server-side error

🔍 Step 10: Understanding Key Annotations

🌱 @SpringBootApplication

Purpose: Main application entry point

Contains: @Configuration + @EnableAutoConfiguration + @ComponentScan

Usage: Only on main class

🌐 @RestController

Purpose: Marks class as REST API controller

Contains: @Controller + @ResponseBody

Effects: Automatic JSON serialization

🛠️ @Service

Purpose: Marks class as business logic service

Effects: Spring component scanning and registration

Best Practice: Contains business logic, not web logic

🗺️ @RequestMapping

Purpose: Maps HTTP requests to methods

Variants: @GetMapping, @PostMapping, @PutMapping, @DeleteMapping

Parameters: path, method, headers, params

📨 @RequestBody

Purpose: Deserializes JSON to Java object

Usage: POST/PUT method parameters

Effect: Automatic JSON → Object conversion

🛣️ @PathVariable

Purpose: Extracts values from URL path

Example: /api/books/{id} → @PathVariable Long id

Usage: RESTful URL parameters

🚨 Common Issues and Solutions

🔴 Port 8080 Already in Use

Error: "Port 8080 was already in use"

Solution: Change port in application.properties:

server.port=8081

🔴 404 - API Not Found

Issue: Endpoints return 404

Check: URL mapping and controller annotations

Solution: Verify @RestController and @RequestMapping

🔴 JSON Serialization Errors

Issue: Objects not converting to JSON

Solution: Add getters/setters and default constructor

🔴 Dependency Injection Fails

Issue: @Autowired not working

Solution: Check @Service, @Component annotations and package scanning
