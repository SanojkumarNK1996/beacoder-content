Links and Images

🎯 Why This Exists (Indian Real-Life Example)

  • Every product card on Flipkart or Amazon is a combination of an image and a link.
  • News websites use links to navigate to full articles and images to attract attention.

📖 What Is This

  • Links (<a>) let users navigate between pages or download resources.
  • Images (<img>) display pictures, icons, and banners on a page.

🧾 Syntax / Basic Usage ⭐

<a href="https://www.example.com">Visit Example</a>

<img src="logo.png" alt="Site Logo" />

<a href="product.html">
  <img src="mobile.png" alt="Mobile Phone" />
</a>

🔍 How It Works (Step-by-Step Execution) ⭐

  1. <a> creates a clickable area; href specifies the target URL.
  2. Clicking the link asks the browser to load the target page.
  3. <img> fetches an image from the path given in src and renders it inline.
  4. alt text is used by screen readers and displayed if image fails to load.

When to Use

  • Links: navigation menus, buttons, references, “Read more”.
  • Images: logos, product photos, thumbnails, icons.

When NOT to Use

  • Don’t use large images where simple icons or text would do.
  • Don’t make non-obvious UI elements into links without clear affordance.

Minimum You MUST Remember

  • <a href="URL">Text</a> for hyperlinks.
  • <img src="path" alt="description" /> for images.
  • Use target="_blank" carefully (new tab) and always with rel="noopener" for security.

Common Mistakes

  • Incorrect/relative paths causing broken links or images.
  • Missing alt text, which hurts accessibility and SEO.
  • Nesting one <a> tag inside another (invalid HTML).

Interviewer’s Perspective

  • How do you make an image clickable?
  • Why is the alt attribute important?
