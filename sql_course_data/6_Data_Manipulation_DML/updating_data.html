Updating Data (UPDATE): 10-Part Beginner-Friendly Guide

1. Why Does This Topic Exist?

In real life, data changes—students get new marks, customers update their address, or an order status changes. The UPDATE command lets you modify existing records in your database.

2. What Is This?

UPDATE is an SQL command used to change values in one or more columns for existing rows in a table.

3. Syntax/Structure

-- Update specific columns for matching rows
UPDATE table_name
SET column1 = value1, column2 = value2
WHERE condition;

4. Step-by-Step: How It Works

  1. Decide which table and rows you want to update.
  2. Write the UPDATE statement with SET for new values and WHERE for the condition.
  3. Run the command in your SQL tool.
  4. Check your table with SELECT to confirm the update.

5. When to Use

  • When a student’s marks change after revaluation.
  • When a customer updates their phone number.
  • When you want to change any existing record.

6. When Not to Use

  • When you want to add new data (use INSERT instead).
  • When you want to remove data (use DELETE instead).

7. Minimum to Remember

  • Always use WHERE to avoid updating all rows by mistake.
  • You can update one or many columns at once.
  • Use expressions in SET (e.g., Age = Age + 1).

8. Common Mistakes

  • Forgetting WHERE (updates every row!).
  • Misspelling column names.
  • Setting wrong values or data types.

9. Interviewer’s Perspective

  • “How do you update only one record in SQL?”
  • “What happens if you omit WHERE in UPDATE?”
  • “How can you increase all employees’ salaries by 10%?”

10. Practice Questions

  1. Write SQL to update the address of a customer with CustomerID = 5.
  2. How would you set all students’ attendance to 100% in the Students table?
  3. What happens if you run UPDATE Students SET Age = 20; without WHERE?

8️⃣ Viewing Updated Data

After updating records, you can use the SELECT statement to confirm the changes:

SELECT * FROM Students WHERE StudentID = 101;

This helps you verify that your update worked as intended.

9️⃣ Example Scenario

Suppose you have a Students table like this:

StudentIDNameAgeGrade
101John19B
102Mary18B

You can update John’s grade like this:

UPDATE Students
SET Grade = 'A'
WHERE Name = 'John';

Now John’s record looks like this:

StudentIDNameAgeGrade
101John19A
102Mary18B

🔟 Summary

  • UPDATE table SET column=value WHERE condition; — modify existing data.
  • Always use WHERE to prevent accidental updates to all rows.
  • You can update multiple columns at once.
  • Expressions can be used for calculations (e.g., Age = Age + 1).
  • Use SELECT after updating to verify your results.
