Updating Data (UPDATE)

The UPDATE statement is used to modify existing records in a table. You can update one or more columns, and you can also control which rows are updated using a WHERE clause.

1️⃣ Basic UPDATE Syntax

UPDATE table_name
SET column1 = value1, column2 = value2, ...
WHERE condition;

This command updates specific columns for the rows that match the given condition.

2️⃣ Example: Updating One Column

UPDATE Students
SET Age = 20
WHERE Name = 'John';

This command changes John’s age to 20 — only for the record where Name equals 'John'.

3️⃣ Example: Updating Multiple Columns

UPDATE Students
SET Age = 21, Grade = 'A'
WHERE StudentID = 101;

This changes both the Age and Grade columns for the student whose ID is 101.

4️⃣ Important: Always Use WHERE!

If you don’t include a WHERE clause, all rows in the table will be updated.

UPDATE Students
SET Grade = 'B';

This will set the Grade to 'B' for every student in the table. Always double-check your WHERE condition before running an UPDATE!

5️⃣ Updating with Expressions

You can also use mathematical expressions when updating values.

UPDATE Students
SET Age = Age + 1
WHERE Class = '10A';

This increases the Age of all students in class 10A by 1.

6️⃣ Using UPDATE with Conditions

You can combine multiple conditions using AND and OR.

UPDATE Students
SET Grade = 'A'
WHERE Age > 18 AND Attendance > 90;

This updates the Grade to 'A' for students older than 18 who have over 90% attendance.

7️⃣ Updating from Another Table (Advanced)

Sometimes, you may need to update values based on another table’s data. This is often done using a JOIN or a subquery.

Example (MySQL style):

UPDATE Students s
JOIN Classes c ON s.ClassID = c.ClassID
SET s.TeacherName = c.ClassTeacher
WHERE c.ClassName = '10B';

This updates each student’s TeacherName based on data from the Classes table.

8️⃣ Viewing Updated Data

After updating records, you can use the SELECT statement to confirm the changes:

SELECT * FROM Students WHERE StudentID = 101;

This helps you verify that your update worked as intended.

9️⃣ Example Scenario

Suppose you have a Students table like this:

StudentIDNameAgeGrade
101John19B
102Mary18B

You can update John’s grade like this:

UPDATE Students
SET Grade = 'A'
WHERE Name = 'John';

Now John’s record looks like this:

StudentIDNameAgeGrade
101John19A
102Mary18B

🔟 Summary

  • UPDATE table SET column=value WHERE condition; — modify existing data.
  • Always use WHERE to prevent accidental updates to all rows.
  • You can update multiple columns at once.
  • Expressions can be used for calculations (e.g., Age = Age + 1).
  • Use SELECT after updating to verify your results.
