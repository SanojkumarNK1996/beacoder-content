Inserting Data (INSERT): 10-Part Beginner-Friendly Guide

1. Why Does This Topic Exist?

Every real-world app—like Paytm, Zomato, or your college ERP—needs to add new data (users, orders, marks) to its database. The INSERT command is how you add new records to your tables.

2. What Is This?

INSERT is an SQL command used to add new rows (records) to a table. It’s how you put real data into your database.

3. Syntax/Structure

-- Insert into specific columns
INSERT INTO table_name (column1, column2, ...)
VALUES (value1, value2, ...);

-- Insert into all columns (order must match table)
INSERT INTO table_name
VALUES (value1, value2, ...);

-- Insert multiple rows at once
INSERT INTO table_name (column1, column2)
VALUES (1, 'Amit'), (2, 'Priya');

4. Step-by-Step: How It Works

  1. Decide which table you want to add data to.
  2. Write the INSERT INTO statement with the correct columns and values.
  3. Run the command in your SQL tool (like MySQL Workbench, DBeaver, or command line).
  4. Check your table with SELECT * FROM table_name; to see the new data.

5. When to Use

  • When a new user signs up (insert into Users table).
  • When a new order is placed (insert into Orders table).
  • When you want to add any new record to your system.

6. When Not to Use

  • When you want to update existing data (use UPDATE instead).
  • When you want to remove data (use DELETE instead).

7. Minimum to Remember

  • Always match the number of columns and values.
  • Use column names for clarity.
  • Use NULL for empty values if allowed.
  • You can insert multiple rows in one command.

8. Common Mistakes

  • Mismatching column count and values.
  • Violating constraints (e.g., duplicate PRIMARY KEY).
  • Wrong data type (e.g., text in numeric columns).

9. Interviewer’s Perspective

  • “How do you insert multiple rows in one SQL statement?”
  • “What happens if you omit a NOT NULL column in INSERT?”
  • “Why is it better to specify column names in INSERT?”

10. Practice Questions

  1. Write SQL to insert a new student (ID: 10, Name: Ravi, Age: 21) into the Students table.
  2. How would you insert two new products into a Products table in one command?
  3. What happens if you try to insert a duplicate PRIMARY KEY?

You can insert data from another table using the SELECT statement.

INSERT INTO Graduates (ID, Name)
SELECT StudentID, Name FROM Students WHERE Age > 20;

This copies all students older than 20 into the Graduates table.

8️⃣ Viewing Inserted Data

After inserting, use the SELECT command to check what’s inside your table:

SELECT * FROM Students;

This displays all rows in the Students table.

9️⃣ Example: Putting It All Together

CREATE TABLE Students (
  StudentID INT PRIMARY KEY,
  Name VARCHAR(50) NOT NULL,
  Age INT
);

INSERT INTO Students (StudentID, Name, Age)
VALUES 
(1, 'Alice', 18),
(2, 'Bob', 20),
(3, 'Charlie', 19);

SELECT * FROM Students;

Output:

+-----------+---------+-----+
| StudentID | Name    | Age |
+-----------+---------+-----+
| 1         | Alice   | 18  |
| 2         | Bob     | 20  |
| 3         | Charlie | 19  |
+-----------+---------+-----+

10️⃣ Summary

  • INSERT INTO ... VALUES (...); — adds new rows to a table.
  • Include column names for clarity.
  • Can insert multiple rows in one command.
  • Can use DEFAULT and NULL where applicable.
  • Use SELECT to verify inserted data.
