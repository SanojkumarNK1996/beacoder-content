Aggregate Functions: 10-Part Beginner-Friendly Guide

1. Why Does This Topic Exist?

In real life—banking, shopping, exams—you often need totals, averages, or highest/lowest values. SQL aggregate functions help you quickly summarize large amounts of data, like finding the average marks in a class or total sales in a month.

2. What Is This?

Aggregate functions perform calculations on groups of rows and return a single value. Common ones are COUNT, SUM, AVG, MAX, and MIN.

3. Syntax/Structure

-- Count rows
SELECT COUNT(*) FROM table_name;

-- Sum values
SELECT SUM(column) FROM table_name;

-- Average value
SELECT AVG(column) FROM table_name;

-- Maximum and minimum
SELECT MAX(column), MIN(column) FROM table_name;

4. Step-by-Step: How It Works

  1. Decide what summary you need (count, sum, average, etc.).
  2. Write a SELECT statement with the aggregate function.
  3. Optionally use WHERE to filter which rows to include.
  4. Run the query to get your answer.

5. When to Use

  • To count students who passed an exam.
  • To find the total sales in a shop.
  • To get the highest and lowest marks in a class.

6. When Not to Use

  • When you want to see individual records (use plain SELECT instead).
  • When you need to update or delete data (use UPDATE or DELETE).

7. Minimum to Remember

  • COUNT() counts rows.
  • SUM() adds up values.
  • AVG() finds the average.
  • MAX() and MIN() find the highest and lowest.

8. Common Mistakes

  • Forgetting to use WHERE to filter data before aggregating.
  • Using aggregate functions on text columns (causes errors).
  • Confusing COUNT(*) (all rows) with COUNT(column) (non-null values only).

7️⃣ Combining Multiple Aggregate Functions

You can include several aggregate functions in one query:

SELECT COUNT(*) AS TotalStudents,
       AVG(Marks) AS AverageMarks,
       MAX(Marks) AS TopScore,
       MIN(Marks) AS LowestScore
FROM Students;

This produces a summary of the entire class in one row.

8️⃣ Ignoring NULL Values

Aggregate functions (except COUNT(*)) ignore NULL values automatically.

SELECT AVG(Salary) FROM Employees;

This will calculate the average salary using only non-null values.

9️⃣ Using Aggregate Functions with Aliases

Use AS to rename the output column for better readability.

SELECT COUNT(*) AS Total, AVG(Salary) AS AverageSalary
FROM Employees;

This makes your result easier to understand in reports.

11️⃣ When to Use Aggregate Functions

Aggregate functions are typically used when you want summarized information instead of individual row data, such as:

  • Total number of customers
  • Average sales per month
  • Maximum and minimum salaries
  • Total revenue for a product

12️⃣ Summary

  • COUNT() → number of rows
  • SUM() → total value
  • AVG() → average value
  • MAX() → highest value
  • MIN() → lowest value
