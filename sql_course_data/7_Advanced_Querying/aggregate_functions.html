Aggregate Functions (COUNT, SUM, AVG, MAX, MIN)

Aggregate functions are special SQL functions that perform calculations on a group of rows and return a single value. They are widely used for reporting, analytics, and summarizing data in databases.

1️⃣ What Are Aggregate Functions?

Aggregate functions take multiple values (rows) as input and return one summarized result. For example, you can calculate the total number of students, the average age, or the highest score in a class.

Common aggregate functions in SQL are:

  • COUNT() — counts the number of rows
  • SUM() — adds up numeric values
  • AVG() — calculates the average value
  • MAX() — finds the highest value
  • MIN() — finds the lowest value

2️⃣ COUNT()

The COUNT() function counts the number of rows in a table or the number of non-null values in a column.

SELECT COUNT(*) AS TotalStudents
FROM Students;

This query returns the total number of students in the Students table.

You can also count only specific rows using a condition:

SELECT COUNT(*) AS PassedStudents
FROM Students
WHERE Grade = 'A';

3️⃣ SUM()

The SUM() function adds up all the numeric values in a column.

SELECT SUM(Marks) AS TotalMarks
FROM Students;

This returns the sum of all students’ marks.

You can combine it with conditions:

SELECT SUM(Marks) AS ScienceMarks
FROM Students
WHERE Subject = 'Science';

4️⃣ AVG()

The AVG() function calculates the average (mean) value of a numeric column.

SELECT AVG(Marks) AS AverageMarks
FROM Students;

This gives the average mark of all students.

5️⃣ MAX() and MIN()

The MAX() function returns the highest value, and the MIN() function returns the lowest value in a column.

SELECT MAX(Marks) AS HighestMark, MIN(Marks) AS LowestMark
FROM Students;

This shows both the top and lowest scores in the class.

6️⃣ Using Aggregate Functions with WHERE

You can combine aggregate functions with WHERE to focus on specific groups of data.

SELECT AVG(Marks) AS AverageScience
FROM Students
WHERE Subject = 'Science';

This calculates the average marks for only Science students.

7️⃣ Combining Multiple Aggregate Functions

You can include several aggregate functions in one query:

SELECT COUNT(*) AS TotalStudents,
       AVG(Marks) AS AverageMarks,
       MAX(Marks) AS TopScore,
       MIN(Marks) AS LowestScore
FROM Students;

This produces a summary of the entire class in one row.

8️⃣ Ignoring NULL Values

Aggregate functions (except COUNT(*)) ignore NULL values automatically.

SELECT AVG(Salary) FROM Employees;

This will calculate the average salary using only non-null values.

9️⃣ Using Aggregate Functions with Aliases

Use AS to rename the output column for better readability.

SELECT COUNT(*) AS Total, AVG(Salary) AS AverageSalary
FROM Employees;

This makes your result easier to understand in reports.

11️⃣ When to Use Aggregate Functions

Aggregate functions are typically used when you want summarized information instead of individual row data, such as:

  • Total number of customers
  • Average sales per month
  • Maximum and minimum salaries
  • Total revenue for a product

12️⃣ Summary

  • COUNT() → number of rows
  • SUM() → total value
  • AVG() → average value
  • MAX() → highest value
  • MIN() → lowest value
