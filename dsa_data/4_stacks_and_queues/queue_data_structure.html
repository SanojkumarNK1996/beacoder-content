🚶 Queue Data Structure

🎯 What is a Queue?

A queue follows FIFO — First In, First Out. Elements are added at the rear and removed from the front.

🎟️ Ticket Counter Analogy

People join at the back and leave from the front. First person in line is first served.

🔧 Core Operations

  • enqueue(x) – Add to rear: O(1)
  • dequeue() – Remove from front: O(1)
  • peek() – View front element: O(1)
Queue<Integer> queue = new LinkedList<>();
queue.offer(10);
queue.offer(20);
queue.offer(30);
System.out.println(queue.peek());  // 10
System.out.println(queue.poll());  // 10 (removed)
System.out.println(queue.peek());  // 20

🎯 Summary

Queues are essential for BFS, task scheduling, producer-consumer patterns.
