🔨 Stack Implementation

🧱 Array-based Stack

class ArrayStack {
    private int[] data;
    private int top = -1;

    ArrayStack(int capacity) { data = new int[capacity]; }

    void push(int x) { data[++top] = x; }
    int pop() { return data[top--]; }
    int peek() { return data[top]; }
    boolean isEmpty() { return top == -1; }
}

🔗 Linked List-based Stack

Each node is one element; head is the top. Push = addFirst, Pop = removeFirst — both O(1).

⚖️ Comparison

  • Array: O(1) amortized push (needs resize occasionally), fixed max size or resizable
  • LinkedList: Always O(1) push/pop, extra memory per node

🎯 Summary

In Java, prefer ArrayDeque (array-backed) as it's faster than LinkedList for stack operations in most cases.
