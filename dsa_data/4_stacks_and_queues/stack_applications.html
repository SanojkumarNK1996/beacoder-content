🎯 Stack Applications

📞 Function Call Stack

Every time a method is called, a frame is pushed onto the JVM call stack. On return, the frame is popped.

🔢 Expression Evaluation

Stacks are used to evaluate/parse arithmetic expressions (infix → postfix, balanced parentheses).

// Check balanced parentheses
boolean isBalanced(String s) {
    Deque<Character> stack = new ArrayDeque<>();
    for (char c : s.toCharArray()) {
        if (c == '(' || c == '[' || c == '{') stack.push(c);
        else if (c == ')' && (stack.isEmpty() || stack.pop() != '(')) return false;
        else if (c == ']' && (stack.isEmpty() || stack.pop() != '[')) return false;
        else if (c == '}' && (stack.isEmpty() || stack.pop() != '{')) return false;
    }
    return stack.isEmpty();
}
