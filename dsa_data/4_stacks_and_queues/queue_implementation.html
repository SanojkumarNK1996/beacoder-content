🔨 Queue Implementation

🔄 Circular Queue

Uses a fixed-size array with front/rear indices that wrap around to avoid wasted space.

⚡ Priority Queue

Elements dequeued in priority order (not FIFO). In Java: PriorityQueue.

// Min-heap (smallest dequeued first)
PriorityQueue<Integer> pq = new PriorityQueue<>();
pq.offer(30); pq.offer(10); pq.offer(20);
System.out.println(pq.poll()); // 10 (min)

// Max-heap
PriorityQueue<Integer> maxPQ = new PriorityQueue<>(Collections.reverseOrder());

🔀 Deque (Double-ended Queue)

Supports insert/remove from both ends. In Java: ArrayDeque.

🎯 Summary

Choose LinkedList for simple FIFO, PriorityQueue for priority-based access, and ArrayDeque for deque operations.
