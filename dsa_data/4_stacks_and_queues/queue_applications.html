🎯 Queue Applications

🌐 Breadth-First Search (BFS)

void bfs(int start, Map<Integer, List<Integer>> graph) {
    Queue<Integer> queue = new LinkedList<>();
    Set<Integer> visited = new HashSet<>();
    queue.offer(start);
    visited.add(start);
    while (!queue.isEmpty()) {
        int node = queue.poll();
        System.out.print(node + " ");
        for (int neighbor : graph.getOrDefault(node, List.of())) {
            if (!visited.contains(neighbor)) {
                visited.add(neighbor);
                queue.offer(neighbor);
            }
        }
    }
}

📋 Task Scheduling

OS schedulers maintain ready queues. Thread pool executors (Java) internally use queues for pending tasks.

🔄 Buffer Management

Queues decouple producers and consumers in streaming/IO pipelines.

🎯 Summary

Queues enable level-order processing and are central to BFS, job scheduling, and async communication systems.
