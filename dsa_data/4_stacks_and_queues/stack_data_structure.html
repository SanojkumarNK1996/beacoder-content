📚 Stack Data Structure

🎯 What is a Stack?

A stack is a collection where the last element added is the first one removed (LIFO – Last In, First Out).

🍽️ Plate Analogy

A stack of plates: you always put new plates on top and take from the top. You can't grab from the middle.

🔧 Core Operations

  • push(x) – Add element to top: O(1)
  • pop() – Remove top element: O(1)
  • peek() – View top without removing: O(1)
  • isEmpty() – Check if stack is empty: O(1)
Deque<Integer> stack = new ArrayDeque<>();
stack.push(10);
stack.push(20);
stack.push(30);
System.out.println(stack.peek()); // 30
stack.pop();
System.out.println(stack.peek()); // 20

🎯 Summary

Use Deque (ArrayDeque) in Java as a stack. Stacks are fundamental for recursion simulation, expression evaluation, and backtracking.
