🔍 Searching in Practice

📌 Hash-Based Search

Java's HashMap enables O(1) average lookups. Ideal for: check existence, count frequency, find pairs.

Map<Integer, Integer> freq = new HashMap<>();
for (int n : nums) freq.merge(n, 1, Integer::sum);

🌲 Tree-Based Search

Java's TreeMap / TreeSet (Red-Black Tree) gives O(log n) search + sorted order + range queries via floorKey(), ceilingKey(), subMap().

🎯 Two-Pointer Search

On a sorted array, use two pointers (left and right) to find pairs or triplets in O(n) instead of O(n²).

🎯 Summary

Real-world search combines data structures: HashMap for O(1) lookup, TreeMap for ordered lookup, two-pointer for sorted-array problems.
