🔍 Introduction to Searching

🎯 What is Searching?

Searching is the process of finding a specific element or value in a data structure.

📋 Types

  • Linear Search: Check each element one-by-one. O(n). Works on unsorted data.
  • Binary Search: Divide-and-conquer on sorted data. O(log n).
  • Hash-based Search: O(1) average using a hash table.
  • Tree-based Search: O(log n) on a balanced BST.

⚖️ Choosing a Search Strategy

  • Unsorted, small data → Linear Search
  • Sorted array → Binary Search
  • Frequent lookups by key → Hash Map
  • Range queries / ordered operations → BST / TreeMap

🎯 Summary

Searching is about matching the right data structure and algorithm to the access pattern of your problem.
