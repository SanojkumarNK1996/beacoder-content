🌐 Other Search Algorithms

📈 Interpolation Search

Like binary search but uses the value distribution to estimate position. Better than binary search for uniformly distributed sorted data: O(log log n) average.

🏃 Jump Search

Jump ahead by √n steps, then do linear search backward. O(√n). Works on sorted arrays without random access being too expensive.

📌 Exponential Search

Find range by doubling index, then binary search within it. Useful when the target is near the beginning of an unbounded/infinite array. O(log n).

📋 Search in 2D Matrix

In a row-sorted + column-sorted matrix, start from top-right; go left if too big, down if too small. O(m + n).

🎯 Summary

Binary search is the workhorse. These variants optimize specific scenarios — interpolation, jump, exponential, and matrix search all have their niche.
