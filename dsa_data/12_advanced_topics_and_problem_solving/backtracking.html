🔍 Backtracking Algorithms

🎯 What is Backtracking?

Backtracking is a systematic way to explore all potential solutions by building a solution incrementally and abandoning (backtracking) a path as soon as it violates constraints.

📋 Template

void backtrack(State state, List<State> result) {
    if (isSolution(state)) { result.add(new State(state)); return; }
    for (Choice choice : getChoices(state)) {
        if (isValid(state, choice)) {
            applyChoice(state, choice);
            backtrack(state, result);
            undoChoice(state, choice);  // KEY: undo
        }
    }
}

📌 Classic Problems

  • N-Queens: Place N queens on NxN board with no attacks
  • Sudoku Solver: Fill in digits 1-9 with row/col/box constraints
  • Permutations: Generate all arrangements of a set
  • Subsets: Generate all subsets of a set

🎯 Summary

Backtracking = recursion + pruning. It explores all possibilities but prunes invalid branches early, making it much faster than brute force.
