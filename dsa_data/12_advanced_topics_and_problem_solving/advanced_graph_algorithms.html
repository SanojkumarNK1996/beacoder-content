🌐 Advanced Graph Algorithms

🔍 Strongly Connected Components (SCC)

In a directed graph, an SCC is a maximal set of vertices where every vertex can reach every other. Algorithms: Kosaraju's or Tarjan's (O(V+E)).

🌉 Bridges and Articulation Points

A bridge is an edge whose removal disconnects the graph. An articulation point is a vertex whose removal disconnects the graph. Both found via DFS with low-link values (Tarjan's algorithm).

🔀 Topological Sort (BFS-based: Kahn's Algorithm)

In a DAG, repeatedly remove vertices with in-degree 0. Also detects cycles: if result doesn't include all vertices, a cycle exists.

🔢 Network Flow (Ford-Fulkerson)

Find maximum flow from source to sink. Used in matching, scheduling, circulation problems. O(V × E²) with Edmonds-Karp (BFS-based).

🎯 Summary

Advanced graph algorithms tackle harder problems: SCCs for web crawling / compiler analysis, bridges for network reliability, max-flow for assignment and scheduling.
