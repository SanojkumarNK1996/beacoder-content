📦 Array Fundamentals

🎯 What is an Array?

An array is a collection of elements stored in contiguous memory, each accessible by an index.

int[] numbers = {10, 20, 30, 40, 50};
System.out.println(numbers[2]); // 30 (O(1) access)

📐 Properties

  • Fixed size (in Java, arrays have a fixed length)
  • O(1) random access by index
  • O(n) insertion/deletion (requires shifting)
  • Homogeneous elements (same type)

🧪 Declaring and Initializing

// Declaration
int[] arr;

// Initialization
arr = new int[5];          // [0, 0, 0, 0, 0]
int[] arr2 = {1, 2, 3};   // [1, 2, 3]

🎯 Summary

Arrays are the simplest and most fundamental data structure. They shine for indexed access but are poor for frequent insertions or deletions in the middle.
