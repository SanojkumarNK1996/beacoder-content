🧮 Multi-dimensional Arrays

🎯 2D Arrays (Matrix)

int[][] matrix = {
    {1, 2, 3},
    {4, 5, 6},
    {7, 8, 9}
};

System.out.println(matrix[1][2]); // 6

🔁 Traversing a 2D Array

for (int i = 0; i < matrix.length; i++) {
    for (int j = 0; j < matrix[i].length; j++) {
        System.out.print(matrix[i][j] + " ");
    }
    System.out.println();
}

📌 Common Uses

  • Matrix operations (image processing, game boards)
  • Dynamic programming tables
  • Adjacency matrices for graphs

🎯 Summary

Multi-dimensional arrays represent table/grid data. In Java, they are arrays of arrays and can even be jagged (different row lengths).
