⚙️ Array Operations

🔍 Traversal – O(n)

for (int i = 0; i < arr.length; i++) {
    System.out.println(arr[i]);
}

➕ Insertion

  • At end (if space): O(1)
  • At index i: O(n) — must shift elements right

❌ Deletion

  • At end: O(1)
  • At index i: O(n) — must shift elements left

🔎 Search

  • Unsorted array: O(n) linear search
  • Sorted array: O(log n) binary search

🎯 Summary

Arrays are fast for access, slow for mid-array inserts/deletes. Use ArrayList in Java for dynamic resizing.
