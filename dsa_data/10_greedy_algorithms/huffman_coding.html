📡 Huffman Coding

🎯 What is Huffman Coding?

A lossless data compression algorithm that assigns shorter codes to more frequent characters and longer codes to less frequent ones.

🛠️ Algorithm

  1. Count character frequencies
  2. Build a min-heap of nodes
  3. Repeatedly extract two minimum-frequency nodes, create a parent node with their combined frequency, and insert back
  4. The resulting tree defines codes: left edge = 0, right edge = 1

📐 Complexity

  • Time: O(n log n) where n is the number of distinct characters
  • Always produces an optimal prefix-free code

🎯 Summary

Huffman coding is a classic greedy application. The greedy choice — always merge the two smallest — is provably optimal (exchange argument).
