#️⃣ Hash Tables Fundamentals

🎯 What is a Hash Table?

A hash table maps keys to values using a hash function to compute the index (bucket) in an array where the value is stored.

📐 Complexity

  • Insert: O(1) average
  • Search: O(1) average
  • Delete: O(1) average
  • Worst case: O(n) if all keys hash to the same bucket

🔧 Java Implementation

HashMap<String, Integer> map = new HashMap<>();
map.put("apple", 3);
map.put("banana", 5);
System.out.println(map.get("apple")); // 3
System.out.println(map.containsKey("cherry")); // false

🎯 Summary

Hash tables are the most versatile data structure for fast lookups, frequency counting, and caching. Every programmer must master them.
