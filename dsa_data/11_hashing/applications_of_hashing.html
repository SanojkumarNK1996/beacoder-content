📌 Applications of Hashing

📋 Common Use Cases

  • Frequency counting: Count occurrences of elements in O(n)
  • Two Sum problem: Find pair summing to target in O(n) with a HashSet
  • Anagram detection: Compare character frequency maps
  • Caching / Memoization: Store computed results for fast lookup
  • Deduplication: Use HashSet to track seen elements
  • Grouping: Group elements by key (HashMap<Key, List<Value>>)
// Two Sum - O(n) with HashMap
int[] twoSum(int[] nums, int target) {
    Map<Integer, Integer> map = new HashMap<>();
    for (int i = 0; i < nums.length; i++) {
        int complement = target - nums[i];
        if (map.containsKey(complement)) return new int[]{map.get(complement), i};
        map.put(nums[i], i);
    }
    return new int[]{};
}

🎯 Summary

When you see "find in O(1)" or "check if element exists" in problem constraints, HashSet/HashMap is the answer.
