🌐 Advanced Hashing

🌀 Rolling Hash

A technique to efficiently compute hash values of subarrings/substrings as a sliding window moves. Used in string matching (Rabin-Karp algorithm).

🔐 Cryptographic Hash Functions

MD5, SHA-256 etc. — designed for security (not speed-critical). Properties: one-way (hard to reverse), collision-resistant. Used in: passwords, digital signatures, file integrity.

🎯 Consistent Hashing

Used in distributed systems. Maps both keys and nodes to a hash ring so that adding/removing a node only affects nearby keys — minimizes data movement.

Used in: Apache Cassandra, Amazon DynamoDB, Elasticsearch, CDNs

🗣️ Bloom Filters

A space-efficient probabilistic data structure using multiple hash functions. Answers "possibly in set" or "definitely not in set". Zero false negatives, possible false positives.

🎯 Summary

Advanced hashing techniques solve real distributed-systems problems. Consistent hashing enables scalable distributed caches and databases.
