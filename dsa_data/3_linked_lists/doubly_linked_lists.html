↔️ Doubly Linked Lists

🎯 Structure

Each node has data, a next pointer, and a prev pointer.

class DNode {
    int data;
    DNode next, prev;
    DNode(int data) { this.data = data; }
}

✅ Advantages

  • Can traverse both forward and backward
  • Easier deletion of a node when you have a reference to it (no need to find previous node)

⚠️ Disadvantages

  • Extra memory per node (two pointers vs one)
  • More complex insertion/deletion logic

🎯 Summary

Doubly linked lists are used in LRU caches, browser history, undo/redo — anywhere you need backward traversal.
