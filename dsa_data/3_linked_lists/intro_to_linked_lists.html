🔗 Introduction to Linked Lists

🎯 What is a Linked List?

A linked list is a sequence of nodes where each node holds data and a reference (pointer) to the next node.

🚂 Train Analogy

Like a train: each carriage (node) is connected to the next with a coupling (pointer). You can add or remove carriages without reshuffling the rest of the train.

📐 Array vs Linked List

OperationArrayLinked List
Access by indexO(1)O(n)
Insert at startO(n)O(1)
Insert at middleO(n)O(n) traversal + O(1) insert
MemoryContiguousScattered (extra pointer overhead)

🎯 Summary

Use linked lists when you need frequent insertions/deletions and don't need random access by index.
