➡️ Singly Linked Lists

🎯 Structure

Each node has data and a next pointer. The last node points to null.

class Node {
    int data;
    Node next;
    Node(int data) { this.data = data; }
}

class SinglyLinkedList {
    Node head;

    // Add to front
    void addFirst(int data) {
        Node node = new Node(data);
        node.next = head;
        head = node;
    }

    // Traverse
    void print() {
        Node cur = head;
        while (cur != null) {
            System.out.print(cur.data + " -> ");
            cur = cur.next;
        }
        System.out.println("null");
    }
}

🎯 Summary

Singly linked lists are simple and memory-efficient. You can only traverse in one direction.
