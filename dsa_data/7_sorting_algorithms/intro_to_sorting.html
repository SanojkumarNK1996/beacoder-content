📊 Introduction to Sorting

🎯 What is Sorting?

Sorting arranges elements in a defined order (ascending or descending). It is one of the most fundamental operations in computer science.

📋 Classification

  • Comparison-based: Compares elements (Bubble, Merge, Quick). Lower bound: O(n log n)
  • Non-comparison: Uses element properties (Counting, Radix, Bucket). Can beat O(n log n)
  • In-place: O(1) extra space (Quick, Heap Sort)
  • Stable: Preserves relative order of equal elements (Merge, Bubble)

⚖️ Why It Matters

Sorting enables binary search, simplifies duplicate detection, and is a preprocessing step in many algorithms (greedy, DP).

🎯 Summary

Knowing which algorithm to choose — based on size, stability, and space constraints — is a key skill.
