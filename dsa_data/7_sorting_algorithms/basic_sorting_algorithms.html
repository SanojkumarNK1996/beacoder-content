🫧 Bubble, Selection & Insertion Sort

🫧 Bubble Sort

Repeatedly swaps adjacent elements that are out of order. O(n²) — only useful for educational purposes.

🔢 Selection Sort

Finds the minimum element and places it at the correct position each pass. O(n²) — not stable.

📌 Insertion Sort

Builds the sorted array one element at a time by inserting each into its correct position.

  • O(n²) worst case, O(n) best case (nearly sorted input)
  • Stable, in-place — great for small or nearly-sorted arrays
void insertionSort(int[] arr) {
    for (int i = 1; i < arr.length; i++) {
        int key = arr[i], j = i - 1;
        while (j >= 0 && arr[j] > key) arr[j + 1] = arr[j--];
        arr[j + 1] = key;
    }
}

🎯 Summary

These O(n²) algorithms are simple but slow. Use insertion sort for small arrays — Java's Arrays.sort() uses it as a base case.
