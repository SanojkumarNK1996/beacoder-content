🏔️ Heap Sort & Comparison

🎯 Heap Sort

Builds a max-heap, then repeatedly extracts the maximum to sort in place.

  • Time: O(n log n) — always
  • Space: O(1) — in-place
  • Not stable

🎯 Summary

Heap Sort guarantees O(n log n) in-place. Use it when memory is critical but stability doesn't matter.
