🎒 Knapsack Problem

🎯 Problem Statement

Given items with weights and values and a knapsack of capacity W, choose items to maximize total value without exceeding capacity.

📋 0/1 Knapsack (Each item used at most once)

int knapsack(int[] wt, int[] val, int W) {
    int n = wt.length;
    int[][] dp = new int[n + 1][W + 1];
    for (int i = 1; i <= n; i++)
        for (int w = 0; w <= W; w++) {
            dp[i][w] = dp[i-1][w];
            if (wt[i-1] <= w) dp[i][w] = Math.max(dp[i][w], val[i-1] + dp[i-1][w - wt[i-1]]);
        }
    return dp[n][W];
}

📌 Unbounded Knapsack

Each item can be used multiple times. Change dp[i-1][w - wt[i-1]] to dp[i][w - wt[i-1]].

🎯 Summary

Knapsack is the canonical DP problem. Its pattern (include/exclude decisions with a capacity constraint) applies to coin change, subset sum, and partition problems.
