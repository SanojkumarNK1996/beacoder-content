🔢 DP: Fibonacci & Memoization

🎯 Classic Problem: Fibonacci

Naive recursion is O(2ⁿ). DP reduces it to O(n).

// Top-Down (Memoization)
Map<Integer, Long> memo = new HashMap<>();
long fib(int n) {
    if (n <= 1) return n;
    if (memo.containsKey(n)) return memo.get(n);
    long result = fib(n - 1) + fib(n - 2);
    memo.put(n, result);
    return result;
}

// Bottom-Up (Tabulation)
long fibTable(int n) {
    long[] dp = new long[n + 1];
    dp[1] = 1;
    for (int i = 2; i <= n; i++) dp[i] = dp[i-1] + dp[i-2];
    return dp[n];
}

🎯 Summary

Fibonacci is the "Hello World" of DP. The memoization pattern — cache and return — applies to hundreds of DP problems.
