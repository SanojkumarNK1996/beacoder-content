💡 DP Problem Patterns

📋 Common DP Patterns

  • Linear DP: dp[i] depends on dp[i-1] or dp[i-k] — Fibonacci, House Robber, Climbing Stairs
  • 2D DP: dp[i][j] — LCS, Edit Distance, Matrix Chain Multiplication
  • Interval DP: dp[i][j] on subarrays — Matrix Chain, Burst Balloons
  • Bitmask DP: dp[mask] for subset enumeration — TSP, covering problems
  • Tree DP: DP on tree nodes — Diameter, Maximum Independent Set on Tree

🎯 Interview Strategy

  1. Identify if it has optimal substructure and overlapping subproblems
  2. Define the state (what dp[i] or dp[i][j] represents)
  3. Write the recurrence relation
  4. Handle base cases
  5. Optimize space if needed (often 1D array instead of 2D)

🎯 Summary

Most DP problems fit a known pattern. Identify the pattern, define the state clearly, and the rest follows.
