💡 Introduction to Dynamic Programming

🎯 What is DP?

Dynamic Programming solves complex problems by breaking them into overlapping subproblems and storing results to avoid recomputation.

📋 Two Key Properties

  1. Optimal substructure: The optimal solution can be built from optimal solutions of subproblems
  2. Overlapping subproblems: The same subproblems are solved multiple times in a recursive approach

🔁 Approaches

  • Top-Down (Memoization): Recursive with a cache to store solved subproblems
  • Bottom-Up (Tabulation): Iterative, build the solution from base cases up

🎯 Summary

If your recursion solves the same subproblems multiple times, apply DP: either memoize the recursion or convert to a bottom-up table.
