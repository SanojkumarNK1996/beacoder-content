🌳 Introduction to Trees

🎯 What is a Tree?

A tree is a hierarchical data structure with a root node and subtrees of children, with no cycles.

📐 Terminology

  • Root – Top node with no parent
  • Node – Element in the tree
  • Leaf – Node with no children
  • Height – Longest path from root to leaf
  • Depth – Distance from root to a node

📋 Types of Trees

  • Binary Tree – Each node has at most 2 children
  • Binary Search Tree (BST) – Ordered binary tree
  • AVL / Red-Black Tree – Self-balancing BST
  • Heap – Complete binary tree with heap property
  • Trie – Prefix tree for strings

🎯 Summary

Trees represent hierarchical relationships: file systems, org charts, XML, and many more.
