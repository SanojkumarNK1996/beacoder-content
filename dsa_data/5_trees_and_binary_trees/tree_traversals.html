🔁 Tree Traversals

🎯 Types

  • Inorder (Left → Root → Right) – Gives sorted output for BST
  • Preorder (Root → Left → Right) – Used to copy/serialize a tree
  • Postorder (Left → Right → Root) – Used to delete/free a tree
  • Level-Order (BFS) – Visit level by level using a queue
void inorder(TreeNode node) {
    if (node == null) return;
    inorder(node.left);
    System.out.print(node.val + " ");
    inorder(node.right);
}

🎯 Summary

Choose traversal based on goal: inorder for sorted sequence, level-order for shortest path, postorder for deletion.
