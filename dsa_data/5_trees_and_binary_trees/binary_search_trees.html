🔍 Binary Search Trees (BST)

🎯 BST Property

For every node: all values in the left subtree < node, all values in the right subtree > node.

🔧 Operations

  • Search: O(log n) avg, O(n) worst
  • Insert: O(log n) avg
  • Delete: O(log n) avg
TreeNode search(TreeNode root, int target) {
    if (root == null || root.val == target) return root;
    if (target < root.val) return search(root.left, target);
    return search(root.right, target);
}

⚠️ Pitfall: Degradation

If you insert already-sorted data, a BST degrades to a linked list (O(n) operations). Use a balanced BST (AVL, Red-Black) to avoid this.

🎯 Summary

BSTs provide O(log n) search on average and maintain sorted order. Ensure balance for consistent performance.
