📊 Graph Representation

🧮 Adjacency Matrix

A 2D array where matrix[i][j] = 1 (or weight) if there's an edge from i to j.

  • Space: O(V²)
  • Check if edge exists: O(1)
  • Best for dense graphs

📋 Adjacency List

Each vertex holds a list of its neighbors.

Map<Integer, List<Integer>> graph = new HashMap<>();
graph.put(0, List.of(1, 2));
graph.put(1, List.of(0, 3));
graph.put(2, List.of(0));
graph.put(3, List.of(1));
  • Space: O(V + E)
  • Best for sparse graphs (most real-world graphs)

🎯 Summary

Prefer adjacency list for most problems. Use adjacency matrix when you need O(1) edge lookups and the graph is dense.
