🌿 Minimum Spanning Tree

🎯 What is an MST?

A Minimum Spanning Tree of a weighted undirected graph is a subset of edges that connects all vertices with the minimum total weight, with no cycles.

🛠️ Kruskal's Algorithm

  • Sort all edges by weight
  • Add edges greedily; skip if it creates a cycle (use Union-Find)
  • Time: O(E log E)

🌱 Prim's Algorithm

  • Start from any vertex, greedily grow the MST by adding the minimum-weight edge connecting the tree to a new vertex
  • Time: O((V + E) log V) with a priority queue

🎯 Summary

Use MST for network design problems (minimum cost to connect all cities, laying cables, etc.).
