🔍 Graph Traversal

🔁 Depth-First Search (DFS)

Explore as deep as possible down each branch before backtracking. Uses a stack (or recursion).

void dfs(int node, Set<Integer> visited, Map<Integer, List<Integer>> graph) {
    visited.add(node);
    System.out.print(node + " ");
    for (int neighbor : graph.getOrDefault(node, List.of())) {
        if (!visited.contains(neighbor)) dfs(neighbor, visited, graph);
    }
}

🌊 Breadth-First Search (BFS)

Explore all neighbors at current depth before going deeper. Uses a queue. Finds shortest path in unweighted graphs.

⚖️ DFS vs BFS

  • DFS: Better for cycle detection, path existence, topological sort
  • BFS: Better for shortest path, level-order exploration

🎯 Summary

Both run in O(V + E). Choose DFS for deep exploration, BFS for shortest paths and layer-by-layer processing.
