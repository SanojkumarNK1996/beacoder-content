🛣️ Shortest Path Algorithms

🎯 Dijkstra's Algorithm

Finds shortest paths from a source to all vertices in a weighted, non-negative graph. Uses a min-heap (PriorityQueue).

  • Time: O((V + E) log V)
  • Does NOT work with negative edge weights

🌐 Floyd-Warshall

Finds shortest paths between all pairs of vertices. Uses dynamic programming.

  • Time: O(V³) — suitable for small dense graphs
  • Handles negative edges (but not negative cycles)

🔄 Bellman-Ford

Single-source shortest paths, handles negative edge weights.

  • Time: O(V × E) — slower than Dijkstra
  • Can detect negative cycles

🎯 Summary

  • Use Dijkstra for most weighted shortest-path problems
  • Use Bellman-Ford if negative weights exist
  • Use Floyd-Warshall for all-pairs shortest paths on small graphs
