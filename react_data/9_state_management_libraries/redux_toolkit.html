🛠️ Redux Toolkit

🎯 What is Redux Toolkit?

Redux Toolkit (RTK) is the official, opinionated toolset for Redux. It eliminates boilerplate and includes best practices by default.

npm install @reduxjs/toolkit react-redux

🔧 Creating a Slice

// cartSlice.js
import { createSlice } from '@reduxjs/toolkit';

const cartSlice = createSlice({
  name: 'cart',
  initialState: { items: [], total: 0 },
  reducers: {
    addItem: (state, action) => {
      state.items.push(action.payload);       // Immer allows "mutations"
      state.total += action.payload.price;
    },
    removeItem: (state, action) => {
      state.items = state.items.filter(i => i.id !== action.payload);
    },
  },
});

export const { addItem, removeItem } = cartSlice.actions;
export default cartSlice.reducer;

🎯 Summary

RTK makes Redux enjoyable. Use createSlice for reducers/actions, configureStore for the store. RTK Query is an extension of RTK for data fetching.
