🐻 Zustand State Management

🎯 What is Zustand?

Zustand is a small, fast, and unopinionated state manager for React. Minimal boilerplate, no Provider needed, works outside React too.

npm install zustand

🔧 Creating a Store

import { create } from 'zustand';

const useCartStore = create((set, get) => ({
  items: [],
  total: 0,

  addItem: (item) => set(state => ({
    items: [...state.items, item],
    total: state.total + item.price,
  })),

  removeItem: (id) => set(state => ({
    items: state.items.filter(i => i.id !== id),
  })),

  getItemCount: () => get().items.length,  // get() reads current state
}));

// Usage — no Provider needed!
function CartIcon() {
  const items = useCartStore(state => state.items);        // Subscribed to items only
  const addItem = useCartStore(state => state.addItem);
  return <span>{items.length} items</span>;
}

🎯 Summary

Zustand is the pragmatic choice for new projects. Zero boilerplate, no Provider, fine-grained subscriptions. Choose Redux Toolkit for large teams needing strict conventions and devtools.
