🎨 React Styling Methods

📋 1. CSS Stylesheets

// Button.css
.btn { background: blue; color: white; padding: 8px 16px; }
.btn-large { font-size: 1.2rem; }

// Button.jsx
import './Button.css';
const Button = () => <button className="btn btn-large">Click</button>;

🔒 2. CSS Modules (Scoped Styles)

// Button.module.css
.btn { background: blue; }

// Button.jsx
import styles from './Button.module.css';
const Button = () => <button className={styles.btn}>Click</button>;
// Class names are auto-scoped: becomes ".Button_btn__abc123"

💅 3. Inline Styles

const style = { background: 'blue', color: 'white', fontSize: '16px' };
<button style={style}>Click</button>

🌊 4. Tailwind CSS (Most Popular Today)

<button className="bg-blue-500 text-white px-4 py-2 rounded hover:bg-blue-600">
  Click
</button>

🎯 Summary

CSS Modules for scoped styles, Tailwind for utility-first rapid development. Both are excellent for React. Avoid inline styles for anything beyond truly dynamic values.
