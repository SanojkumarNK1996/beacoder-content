🏛️ Class Components

🎯 Legacy but Important

Before React Hooks (pre-2019), all stateful components were class components. You'll still encounter them in older codebases.

import { Component } from 'react';

class Counter extends Component {
  constructor(props) {
    super(props);
    this.state = { count: 0 };
  }

  render() {
    return (
      <div>
        <p>Count: {this.state.count}</p>
        <button onClick={() => this.setState({ count: this.state.count + 1 })}>
          Increment
        </button>
      </div>
    );
  }
}

📋 Lifecycle Methods

  • componentDidMount(): After component renders (API calls)
  • componentDidUpdate(prevProps, prevState): After re-render
  • componentWillUnmount(): Before removal (cleanup)

🔄 Equivalent in Hooks

All lifecycle methods are replaced by useEffect() in functional components.

🎯 Summary

Don't write new class components. If you see them in existing code, understand their lifecycle methods, then consider migrating to functional components + hooks.
