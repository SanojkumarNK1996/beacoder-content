📜 Rendering Lists and Conditional Rendering

📋 Rendering Lists with .map()

const fruits = ['Apple', 'Banana', 'Cherry'];

function FruitList() {
  return (
    <ul>
      {fruits.map((fruit, index) => (
        <li key={fruit}>{fruit}</li>  // key is required!
      ))}
    </ul>
  );
}

⚠️ Always provide a unique key prop! React uses keys to efficiently update lists. Use IDs, not array indexes, whenever possible.

🔀 Conditional Rendering

// Ternary operator
{isLoggedIn ? <Dashboard /> : <LoginPage />}

// Short-circuit (render or nothing)
{isLoading && <Spinner />}

// Early return
function UserCard({ user }) {
  if (!user) return <p>No user found</p>;
  return <h2>{user.name}</h2>;
}

🎯 Summary

map for lists (with unique keys). Ternary for if/else rendering. Short-circuit && for optional rendering. Early returns for guard conditions.
