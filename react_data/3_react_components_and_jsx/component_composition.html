🧩 Component Composition

🎯 Building UIs with Composition

React favors composition over inheritance. Build complex UIs by combining small, focused components.

// Small, focused components
const Avatar = ({ src, alt }) => <img src={src} alt={alt} className="avatar" />;

const UserInfo = ({ name, role }) => (
  <div><h3>{name}</h3><span>{role}</span></div>
);

// Composed from smaller parts
const UserCard = ({ user }) => (
  <div className="card">
    <Avatar src={user.avatar} alt={user.name} />
    <UserInfo name={user.name} role={user.role} />
  </div>
);

👶 children Prop

The children prop passes anything between component tags — enables wrapper/container components:

const Card = ({ children, title }) => (
  <div className="card">
    <h2>{title}</h2>
    {children}
  </div>
);

// Usage:
<Card title="My Profile">
  <p>Content goes here</p>
</Card>

🎯 Summary

Compose small components into bigger ones. The children prop is the key to building reusable layout/wrapper components.
