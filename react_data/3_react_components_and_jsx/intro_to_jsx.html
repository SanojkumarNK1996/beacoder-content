📝 Introduction to JSX

🎯 What is JSX?

JSX (JavaScript XML) is a syntax extension that lets you write HTML-like code in JavaScript. React components use JSX to describe the UI. Babel compiles JSX to React.createElement() calls.

📋 JSX Rules

  • Return a single root element (use <div> or <></> Fragment)
  • Close all tags: <img />, <input />
  • Use camelCase for attributes: className not class, onClick not onclick
  • JavaScript expressions in curly braces: {user.name}, {2 + 2}
function Welcome() {
  const name = "Alice";
  const isLoggedIn = true;
  return (
    <>
      <h1 className="header">Hello, {name}!</h1>
      {isLoggedIn && <p>Welcome back.</p>}
    </>
  );
}

🎯 Summary

JSX is not HTML — it's syntactic sugar over JavaScript. The curly braces {} are where JavaScript lives inside your Component's template.
