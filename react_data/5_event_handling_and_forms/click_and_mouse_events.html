🖱️ Handling Click and Mouse Events

🎯 Event Handlers in React

React uses synthetic events — cross-browser wrappers around native DOM events. Handlers are passed as props using camelCase names.

function EventDemo() {
  const handleClick = (e) => {
    e.preventDefault();  // Prevent default browser behavior
    console.log('Clicked!', e.target);
  };

  const handleDoubleClick = () => console.log('Double click!');

  return (
    <div>
      <button onClick={handleClick}>Click me</button>
      <button onDoubleClick={handleDoubleClick}>Double click</button>
      <div
        onMouseEnter={() => console.log('Mouse in')}
        onMouseLeave={() => console.log('Mouse out')}
      >
        Hover me
      </div>
    </div>
  );
}

📌 Passing Arguments to Handlers

// Use arrow function wrapper to pass args:
{items.map(item => (
  <button key={item.id} onClick={() => handleDelete(item.id)}>
    Delete {item.name}
  </button>
))}

🎯 Summary

Events in React are camelCase (onClick), passed as functions (not strings like HTML). Use arrow wrappers to pass arguments in event handlers.
