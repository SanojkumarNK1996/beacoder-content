📋 React Hook Form Library

🎯 Why React Hook Form?

React Hook Form (RHF) is the most popular form library for React. It manages forms with minimal re-renders, built-in validation, and very little code.

npm install react-hook-form

🔧 Basic Usage

import { useForm } from 'react-hook-form';

function LoginForm() {
  const { register, handleSubmit, formState: { errors } } = useForm();

  const onSubmit = (data) => console.log(data);  // Called only if valid

  return (
    <form onSubmit={handleSubmit(onSubmit)}>
      <input
        {...register('email', {
          required: 'Email is required',
          pattern: { value: /\S+@\S+\.\S+/, message: 'Invalid email' }
        })}
      />
      {errors.email && <p>{errors.email.message}</p>}

      <input
        type="password"
        {...register('password', { required: 'Password required', minLength: { value: 8, message: 'Min 8 chars' } })}
      />
      {errors.password && <p>{errors.password.message}</p>}

      <button type="submit">Login</button>
    </form>
  );
}

🎯 Summary

React Hook Form is the go-to for production forms: zero re-renders during typing, easy validation, integrates with Zod/Yup for schema validation. Use it for any form beyond a simple search input.
