📌 useRef and useCallback

🔖 useRef

useRef creates a mutable reference that persists across renders without causing re-renders when changed. Two main uses:

// 1. DOM access
function FocusInput() {
  const inputRef = useRef(null);
  return (
    <>
      <input ref={inputRef} />
      <button onClick={() => inputRef.current.focus()}>Focus</button>
    </>
  );
}

// 2. Store mutable value (no re-render)
function Timer() {
  const intervalRef = useRef(null);
  const start = () => { intervalRef.current = setInterval(tick, 1000); };
  const stop = () => clearInterval(intervalRef.current);
  ...
}

⚡ useCallback

Memoizes a function — returns the same function reference if dependencies haven't changed. Prevents unnecessary re-renders in child components that receive the function as props.

const handleSubmit = useCallback(() => {
  submitForm(data);
}, [data]);   // New function only when data changes

🎯 Summary

Use useRef for DOM access and storing mutable values, useCallback to stabilize function references passed to memoized child components.
