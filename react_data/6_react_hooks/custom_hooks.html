🔧 Custom Hooks

🎯 What are Custom Hooks?

Custom hooks are functions that start with use and call other hooks internally. They let you extract and share stateful logic between components without changing the component tree.

// Custom hook: useFetch
function useFetch(url) {
  const [data, setData] = useState(null);
  const [loading, setLoading] = useState(true);
  const [error, setError] = useState(null);

  useEffect(() => {
    setLoading(true);
    fetch(url)
      .then(r => r.json())
      .then(d => { setData(d); setLoading(false); })
      .catch(e => { setError(e); setLoading(false); });
  }, [url]);

  return { data, loading, error };
}

// Usage in any component:
function UserList() {
  const { data: users, loading, error } = useFetch('/api/users');
  if (loading) return <Spinner />;
  if (error) return <Error msg={error.message} />;
  return <ul>{users.map(u => <li key={u.id}>{u.name}</li>)}</ul>;
}

📌 Common Custom Hooks

  • useLocalStorage(key, initial): Persist state to localStorage
  • useDebounce(value, delay): Debounce rapidly-changing values
  • useWindowSize(): Responsive design helpers
  • useOnClickOutside(ref, callback): Detect clicks outside an element

🎯 Summary

Custom hooks are the primary code-reuse mechanism in React. Extract any logic that uses hooks and is shared across components into a custom hook.
