🔢 Array Methods: map, filter, reduce

🗺️ .map() — Transform Each Element

const nums = [1, 2, 3, 4];
const doubled = nums.map(n => n * 2); // [2, 4, 6, 8]

// In React — render a list:
const users = ['Alice', 'Bob', 'Carol'];
return (
  <ul>
    {users.map(user => <li key={user}>{user}</li>)}
  </ul>
);

🔍 .filter() — Keep Matching Elements

const ages = [15, 22, 18, 30, 16];
const adults = ages.filter(age => age >= 18); // [22, 18, 30]

// Show only active users:
const active = users.filter(u => u.isActive);

➕ .reduce() — Aggregate to Single Value

const nums = [1, 2, 3, 4, 5];
const sum = nums.reduce((acc, n) => acc + n, 0); // 15

// Build an object from array:
const byId = users.reduce((acc, u) => ({...acc, [u.id]: u}), {});

🎯 Summary

map, filter, reduce replace almost all for-loops in modern React code. They're non-mutating (return new arrays) — perfect for React's immutable state pattern.
