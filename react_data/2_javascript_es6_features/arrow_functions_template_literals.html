🏹 Arrow Functions and Template Literals

🏹 Arrow Functions

Concise function syntax introduced in ES6. Used everywhere in React.

// Traditional function
function add(a, b) { return a + b; }

// Arrow function
const add = (a, b) => a + b;

// With body
const greet = (name) => {
  const msg = `Hello, ${name}!`;
  return msg;
};

// Event handler in React
<button onClick={() => setCount(count + 1)}>Click</button>

📝 Template Literals

Backtick strings with embedded expressions: `Hello, ${name}!`

const name = "Alice";
const age = 30;
// Old way:
console.log("Name: " + name + ", Age: " + age);
// Template literal:
console.log(`Name: ${name}, Age: ${age}`);
// Multi-line:
const html = `
  <div>
    <h1>${name}</h1>
  </div>
`;

🎯 Summary

Arrow functions and template literals appear in nearly every React component. Master them — they're the syntax of modern JavaScript.
