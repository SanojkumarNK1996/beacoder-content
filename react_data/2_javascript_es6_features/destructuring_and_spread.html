🔬 Destructuring and Spread Operator

📦 Array Destructuring

const [a, b] = [1, 2];           // a=1, b=2
const [first, ...rest] = [1,2,3]; // first=1, rest=[2,3]

// In React:
const [count, setCount] = useState(0);

🔑 Object Destructuring

const user = { name: 'Alice', age: 30, city: 'London' };
const { name, age } = user;          // name='Alice', age=30
const { name: userName } = user;    // Rename: userName='Alice'

// In React (function props):
function Profile({ name, age }) {   // Destructure props
  return <p>{name}, {age}</p>;
}

✨ Spread Operator

// Spread arrays
const nums = [1, 2, 3];
const more = [...nums, 4, 5];  // [1, 2, 3, 4, 5]

// Spread objects (used in state updates)
const user = { name: 'Alice', age: 30 };
const updated = { ...user, age: 31 }; // Replace age only

🎯 Summary

Destructuring makes props cleaner; spread is essential for immutable state updates in React. Both are used constantly.
