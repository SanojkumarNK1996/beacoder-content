📡 Axios Library

🎯 Why Axios?

Axios is a popular HTTP client library with a cleaner API than Fetch: automatic JSON parsing, request/response interceptors, better error handling, and built-in timeout support.

npm install axios

🔧 Basic Usage

import axios from 'axios';

// GET request
const response = await axios.get('/api/users');
const users = response.data;  // Auto-parses JSON

// POST request
const newUser = await axios.post('/api/users', { name: 'Alice', email: 'alice@example.com' });

🏗️ Axios Instance (Recommended)

// api.js - create a configured instance
const api = axios.create({
  baseURL: 'https://api.myapp.com',
  timeout: 10000,
  headers: { 'Content-Type': 'application/json' }
});

// Request interceptor: add auth token
api.interceptors.request.use(config => {
  config.headers.Authorization = `Bearer ${getToken()}`;
  return config;
});

// Response interceptor: handle errors globally
api.interceptors.response.use(
  response => response,
  error => {
    if (error.response?.status === 401) logout();
    return Promise.reject(error);
  }
);

export default api;

🎯 Summary

Create one Axios instance with your base URL and interceptors. Import that instance everywhere — centralized auth header injection and global error handling.
