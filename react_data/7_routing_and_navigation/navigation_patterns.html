🗺️ Navigation Patterns and Best Practices

📋 Common Navigation Patterns

1. App Shell with Nested Routes

Top-level routes define layouts (Header, Sidebar) using <Outlet />. Child routes render inside the layout.

2. Modal Routing

Show a modal when navigating to a URL (e.g., Instagram-style /photos/:id), fall back to full page if accessed directly.

3. Scroll Restoration

// Scroll to top on route change
function ScrollToTop() {
  const { pathname } = useLocation();
  useEffect(() => { window.scrollTo(0, 0); }, [pathname]);
  return null;
}

🔍 Useful Router Hooks

  • useLocation(): Get current URL pathname, search, state
  • useParams(): Get URL parameters
  • useNavigate(): Programmatic navigation
  • useSearchParams(): Read/write URL query string

🎯 Summary

Design routes to mirror your app's information architecture. Use nested routes for shared layouts. Scroll to top on navigation as a UX best practice.
