🗺️ Introduction to React Router

🎯 What is React Router?

React Router is the standard routing library for React. It enables client-side navigation — the URL changes and the view updates without a full page reload.

npm install react-router-dom

🔧 Basic Setup (v6)

import { BrowserRouter, Routes, Route, Link } from 'react-router-dom';

function App() {
  return (
    <BrowserRouter>
      <nav>
        <Link to="/">Home</Link> | <Link to="/about">About</Link>
      </nav>
      <Routes>
        <Route path="/" element={<HomePage />} />
        <Route path="/about" element={<AboutPage />} />
        <Route path="/users/:id" element={<UserPage />} />
        <Route path="*" element={<NotFoundPage />} />
      </Routes>
    </BrowserRouter>
  );
}

🎯 Key Components

  • <BrowserRouter>: Wraps app, uses HTML5 history API
  • <Routes>: Matches URL to a route
  • <Route>: Defines a URL path → component mapping
  • <Link>: Navigate without page reload (use instead of <a>)

🎯 Summary

React Router v6 is clean and declarative. Define routes with <Route>, navigate with <Link> or useNavigate().
