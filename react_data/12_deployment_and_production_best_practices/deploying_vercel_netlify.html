🚀 Deploying React to Vercel and Netlify

🎯 Vercel (Recommended for React)

Vercel is the easiest deployment platform — zero configuration, automatic deployments on git push, preview URLs for PRs.

# CLI deployment
npm install -g vercel
vercel       # Interactive setup
vercel --prod  # Deploy to production

# Or: Connect GitHub repo at vercel.com — auto-deploys on every push

🌐 Netlify

# netlify.toml — configuration file
[build]
  command = "npm run build"
  publish = "dist"   # or "build" for CRA

# SPA redirect fix (critical for React Router):
[[redirects]]
  from = "/*"
  to = "/index.html"
  status = 200

📋 SPA Routing Fix

React apps use client-side routing — the server needs to serve index.html for ALL routes, letting React Router handle them.

  • Vercel: Automatic for Vite/CRA projects
  • Netlify: Add [[redirects]] in netlify.toml
  • Nginx: try_files $uri $uri/ /index.html

🎯 Summary

Vercel/Netlify provide instant, free React hosting with CDN, automatic HTTPS, and preview deployments. Connect your GitHub repo and deploy in minutes.
