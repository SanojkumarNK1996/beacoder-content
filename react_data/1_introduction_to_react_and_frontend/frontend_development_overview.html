🌐 Frontend Development Overview

🎯 The Frontend Stack

Frontend development means building what users see and interact with in a browser. Three core technologies:

  • HTML: Structure (what's on the page)
  • CSS: Presentation (how it looks)
  • JavaScript: Behavior (how it works)

📋 Modern Frontend

  • SPA (Single Page Application): JavaScript dynamically updates the page without full reloads (React, Vue, Angular)
  • SSR (Server-Side Rendering): HTML generated on the server for each request (Next.js)
  • SSG (Static Site Generation): HTML pre-generated at build time (Gatsby, Next.js)

🔧 Modern Build Tools

  • Vite: Ultra-fast development server and build tool (recommended)
  • Webpack: Traditional bundler used by Create React App
  • npm/yarn/pnpm: Package managers

🎯 Summary

Frontend has evolved from simple HTML pages to complex SPAs. React is the dominant choice for building modern, interactive SPAs.
