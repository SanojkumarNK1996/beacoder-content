🛠️ Setting Up React Environment

📋 Prerequisites

  • Node.js (v18+): Download from nodejs.org
  • npm (bundled with Node) or yarn/pnpm
  • VS Code with React/JSX extensions

🚀 Create a React App (Recommended: Vite)

# Create a new React project with Vite
npm create vite@latest my-app -- --template react
cd my-app
npm install
npm run dev          # Start dev server (http://localhost:5173)

📦 With Create React App (CRA) — Legacy

npx create-react-app my-app
cd my-app
npm start           # http://localhost:3000

🔧 Essential VS Code Extensions

  • ES7+ React/Redux/React-Native snippets
  • Prettier
  • ESLint
  • Auto Import

🎯 Summary

Use Vite for new projects — it's dramatically faster than Create React App for both dev startup and builds.
