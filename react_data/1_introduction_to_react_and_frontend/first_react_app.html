🎉 Your First React App

🎯 Hello World

// src/App.jsx
function App() {
  return (
    <div>
      <h1>Hello, React!</h1>
      <p>My first React component.</p>
    </div>
  );
}

export default App;

📋 Making it Interactive

import { useState } from 'react';

function App() {
  const [count, setCount] = useState(0);

  return (
    <div>
      <h1>Count: {count}</h1>
      <button onClick={() => setCount(count + 1)}>
        Click me!
      </button>
    </div>
  );
}

🔑 Key Observations

  • JSX looks like HTML but is actually JavaScript
  • useState manages component state
  • React re-renders the component when state changes
  • Event handlers are camelCase: onClick, not onclick

🎯 Summary

React = components + state + events = interactive UI. This counter example is the foundation of every React app.
