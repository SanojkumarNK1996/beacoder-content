📁 React Project Structure

📋 Typical Vite + React Structure

my-app/
  public/          # Static assets (favicon, images)
  src/
    assets/        # Images, fonts, etc.
    components/    # Reusable UI components
    pages/         # Page-level components (App routing)
    hooks/         # Custom React hooks
    services/      # API calls
    store/         # State management (Redux/Zustand)
    utils/         # Helper functions
    App.jsx        # Root component
    main.jsx       # Entry point (renders App to DOM)
  index.html       # HTML template
  package.json     # Dependencies and scripts
  vite.config.js   # Build config

🔑 Key Files

  • main.jsx: Entry point — mounts React app to <div id="root"> in index.html
  • App.jsx: Root component, typically holds router and global layout
  • package.json: Lists dependencies; run npm install to install them

🎯 Summary

Organize by feature or by type. For small apps, organize by type (components/, pages/, hooks/). For large apps, organize by feature (feature-x/components/, feature-x/hooks/).
