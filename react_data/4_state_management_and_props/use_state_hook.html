🪝 useState Hook

🎯 Basic Usage

import { useState } from 'react';

function Counter() {
  const [count, setCount] = useState(0);  // initial value = 0

  return (
    <div>
      <p>Count: {count}</p>
      <button onClick={() => setCount(count + 1)}>+</button>
      <button onClick={() => setCount(count - 1)}>-</button>
      <button onClick={() => setCount(0)}>Reset</button>
    </div>
  );
}

📋 Updating Object State

const [user, setUser] = useState({ name: 'Alice', age: 30 });

// Always spread to avoid losing other fields:
setUser({ ...user, age: 31 });

// Functional update (when new state depends on old):
setCount(prevCount => prevCount + 1);

⚠️ Key Rules

  • Never mutate state directly: count++ won't trigger re-render; use setCount(count + 1)
  • State updates are asynchronous — don't read state right after setting it
  • For complex state, consider useReducer

🎯 Summary

useState is the most fundamental hook. Remember: always use the setter function, never mutate directly, spread objects to update.
