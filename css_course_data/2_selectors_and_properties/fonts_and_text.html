Fonts and Text

Why this exists

  • Readable and attractive fonts improve user experience on blogs, news sites, and e-commerce portals.

What is this

  • CSS properties that control fonts, size, weight, line spacing, and alignment.

Syntax / Basic usage

body {
  font-family: Arial, sans-serif;
  font-size: 16px;
}

h1 {
  font-size: 32px;
  font-weight: bold;
  text-align: center;
}

p {
  line-height: 1.5;
}

How it works

  1. font-family defines which font is used, with fallbacks.
  2. font-size controls the size of text.
  3. text-align controls alignment (left, right, center, justify).

Minimum you must remember

  • Always provide fallback fonts (e.g., Arial, sans-serif).
  • Use relative units like rem or em for scalable typography.

Common mistakes

  • Using too many different fonts on one page.
  • Very small font sizes that are hard to read on mobile.

Interviewer perspective

  • Difference between px, em, and rem for font-size?
  • How would you center text inside a block?

Practice questions

  1. Create a heading and paragraph with different fonts and alignments.
  2. Experiment with line-height to improve readability of long text.
