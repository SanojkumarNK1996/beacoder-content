Color and Background

Why this exists

  • Branding for companies (like Paytm blue, SBI blue, Zomato red) depends heavily on color and backgrounds.

What is this

  • CSS properties that control text color, background color, images, and gradients.

Syntax / Basic usage

body {
  background-color: #f5f5f5;
}

h1 {
  color: #333333;
}

section {
  background-image: url('banner.jpg');
  background-size: cover;
}

How it works

  1. color sets the text color of an element.
  2. background-color sets the fill color behind the content.
  3. background-image adds an image, which can be positioned and repeated.

Minimum you must remember

  • Colors can be in named, hex, rgb, or hsl format.
  • Use contrast-friendly colors for readability.

Common mistakes

  • Using very light text on light backgrounds or dark text on dark backgrounds.
  • Large background images without optimization causing slow loading.
