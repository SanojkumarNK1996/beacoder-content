CSS Grid

Why this exists

  • Grid simplifies building complex two-dimensional layouts like dashboards, galleries, and magazine-style pages.

What is this

  • CSS Grid is a powerful layout system for defining rows, columns, and areas.

Syntax / Basic usage

.container {
  display: grid;
  grid-template-columns: 1fr 1fr 1fr;
  gap: 16px;
}

.item {
  background: #eee;
}

How it works

  1. Set display: grid on the parent.
  2. Define columns and rows using grid-template-columns and grid-template-rows.
  3. Place items explicitly or let them auto-flow into the grid.

Minimum you must remember

  • display: grid on container.
  • Use fr units to divide available space.
  • gap creates space between grid items.

Common mistakes

  • Mixing flexbox and grid without a clear plan.
  • Overcomplicating layouts that could be simple.

Interviewer perspective

  • When would you choose grid over flexbox?
  • Explain the fr unit.
