Box Model

Why this exists

  • Every element on a web page behaves like a box; understanding the box model is necessary for proper layout.

What is this

  • The CSS box model describes how margins, borders, padding, and content define the size and space of an element.

Syntax / Basic understanding

div.card {
  width: 300px;
  padding: 16px;
  border: 1px solid #ccc;
  margin: 20px;
}

How it works

  1. Content: actual text or image inside the element.
  2. Padding: space between content and border.
  3. Border: line surrounding the padding and content.
  4. Margin: space outside the border, separating elements.

Minimum you must remember

  • Total element width = content width + padding + border + margin.
  • Use box-sizing: border-box to make sizing easier.

Common mistakes

  • Forgetting that padding and border add to the width/height.
  • Unexpected layouts because of default box-sizing.

Interviewer perspective

  • What does box-sizing: border-box do?
