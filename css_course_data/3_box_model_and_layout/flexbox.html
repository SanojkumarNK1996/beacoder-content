Flexbox

Why this exists

  • To easily build responsive layouts like navbars, card grids, and equal-height columns without complex floats.

What is this

  • Flexbox is a one-dimensional layout system for arranging items in rows or columns.

Syntax / Basic usage

.container {
  display: flex;
  justify-content: space-between;
  align-items: center;
}

.item {
  flex: 1;
}

How it works

  1. Set display: flex on a parent container.
  2. Child elements become flex items.
  3. Use justify-content to control horizontal alignment and spacing.
  4. Use align-items to control vertical alignment.

Minimum you must remember

  • display: flex on the parent, not the children.
  • justify-content (main axis) and align-items (cross axis).

Common mistakes

  • Forgetting that flex direction changes the meaning of main and cross axis.
  • Overusing margins where flex properties would work better.

Interviewer perspective

  • How is flexbox different from CSS grid?
  • Explain justify-content vs align-items.

  2. Build a three-column layout where each column takes equal width.
