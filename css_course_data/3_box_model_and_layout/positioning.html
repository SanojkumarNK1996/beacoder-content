Positioning

Why this exists

  • Positioning allows elements like headers, floating buttons, and sticky navbars to appear exactly where needed.

What is this

  • CSS positioning defines how an element is placed in the document.
  • Main values: static, relative, absolute, fixed, sticky.

Syntax / Basic usage

.box {
  position: relative;
  top: 10px;
  left: 20px;
}

.header {
  position: fixed;
  top: 0;
  width: 100%;
}

How it works

  1. static: default, element follows normal document flow.
  2. relative: moves relative to its original position.
  3. absolute: positioned relative to the nearest positioned ancestor.
  4. fixed: anchored relative to the viewport (screen).
  5. sticky: behaves like relative until a scroll threshold, then sticks.

Minimum you must remember

  • absolute needs a positioned ancestor (not static) to behave predictably.
  • fixed elements stay in place even when the page is scrolled.

Common mistakes

  • Using absolute everywhere and breaking the layout.
  • Not understanding how stacking context and z-index work.
