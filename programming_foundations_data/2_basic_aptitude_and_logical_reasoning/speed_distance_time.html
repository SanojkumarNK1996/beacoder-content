Speed, Distance & Time

This topic links distance traveled, the time taken, and the speed of motion.

Key Formula

Speed = Distance / Time

Examples

  • If a car travels 150 km in 3 hours, speed = 150 / 3 = 50 km/h.
  • If a train moves at 80 km/h for 2.5 hours, distance = 80 × 2.5 = 200 km.

Applications

  • Travel planning
  • Work/time estimation
