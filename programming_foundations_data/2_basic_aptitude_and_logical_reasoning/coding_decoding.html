Coding-Decoding

Coding-decoding problems replace letters or words with symbols, numbers, or other letters using a defined rule.

Common Techniques

  • Substitution: A → Z, B → Y
  • Position shift: each letter moves by a fixed number
  • Word patterns: reverse, alternate, or group letters

Example

If CODE becomes DPEF by shifting each letter forward by 1, then TEST becomes UFTU.
