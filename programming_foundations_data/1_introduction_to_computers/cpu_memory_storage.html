CPU, Memory, Storage

Understanding the processor, memory, and storage is key to knowing how computers work.

CPU (Central Processing Unit)

The CPU executes instructions and performs calculations. It is often called the brain of the computer.

Memory

  • RAM: Random Access Memory stores data temporarily while programs are running.
  • ROM: Read-Only Memory stores permanent instructions used during startup.

Storage

  • HDD: Hard Disk Drive stores data magnetically on spinning disks.
  • SSD: Solid State Drive stores data on flash memory for faster performance.
