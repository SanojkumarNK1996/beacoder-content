Operating Systems Basics

An operating system (OS) manages computer hardware and provides services for applications.

What an OS Does

  • Manages hardware resources like CPU, memory, and storage.
  • Provides a user interface for interacting with the computer.
  • Runs applications and manages their execution.
  • Handles file management and security.

Common Operating Systems

  • Windows: Popular for personal and business use.
  • Linux: Open-source and widely used on servers and desktops.
  • macOS: Designed for Apple Mac computers.
