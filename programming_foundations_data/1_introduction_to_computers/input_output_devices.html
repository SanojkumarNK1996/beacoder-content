Input / Output Devices

Input and output devices allow users to interact with a computer by sending information in and receiving results back.

Input Devices

  • Keyboard: Enters text and commands.
  • Mouse: Controls cursor movement and selections.
  • Scanner: Converts physical documents into digital format.
  • Microphone: Captures audio input.

Output Devices

  • Monitor: Displays visual output from the computer.
  • Printer: Produces physical copies of documents.
  • Speakers: Play audio output.
  • Projector: Shows visual output on a larger screen.
