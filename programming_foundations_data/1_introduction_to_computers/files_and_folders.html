Files and Folders

Files and folders help organize data on a computer so it is easy to find and manage.

Files

Files store information such as documents, images, videos, and programs. Each file has a name and usually a file extension.

Folders

Folders (or directories) group related files together. They create a hierarchical structure for organizing data.

Naming Conventions

  • Use clear, descriptive names.
  • Avoid special characters and spaces where possible.
  • Keep names consistent across related files.
